# 訂正メモ — 追加リターン表現について

作成日：2026-05-02

## 結論

ドリップバッグ追加リターンを検討する際、リポジトリ内の現行リターン・CAMPFIRE掲載文・使用可能表現を確認せずに文面を作らない。

## 訂正内容

以下の表現は、ユーザー確認なしに使わない。

- おみくじ
- 手紙
- 未来手紙
- 梅竹松
- 4分回復
- タイムカプセル

特に「おみくじ」は、現段階では使用不可として扱う。

## 追加リターン設計の前提

追加リターンは、現行CAMPFIREリターンと照合してから決める。

確認する項目：

- 現在のリターン金額
- 現在のリターン内容
- 既存支援者に不公平にならないか
- 使用してよい商品名・表現
- 食品表示、内容量、発送予定
- 送料込み価格として成立するか

## 暫定方針

ドリップバッグ追加リターンは、以下のように仮置きする。

- 名称：ドリップバッグ応援セット
- 価格：現行リターン確認後に決定
- 内容：現行リターンと重複しすぎない構成にする
- 禁止：未確認のギフト文言を足さない

## 今後の対応

チャットでリターン案を出す前に、必ず現行リターン情報を確認する。
不明な場合は、仮案ではなく「現行リターン確認後に判断」と明記する。
