# X投稿上書きログ

このファイルは、巨大な `x-post-log.md` 本体を直接壊さずに、同じ日時の投稿を差し替えるための上書き用ログです。

運用ルール：

- 同じ日時が `x-post-log.md` とこのファイルにある場合、このファイルの投稿を優先する。
- 「おみくじ」は使わない。
- 手紙サービス、未来手紙、梅竹松、4分回復は使わない。
- CAMPFIRE期間中は、旅するコーヒー屋、軽バン移動カフェ、自家焙煎、クラファン拡散を主軸にする。
- ドリップバッグ追加リターンは「器具なしで試せる入口リターン」として扱う。
