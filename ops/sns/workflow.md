# SNS運用フロー

## X（Twitter）投稿フロー

```
1. x-post-log.md に投稿文を追記
2. python x_poster.py --dry-run で確認
3. python x_poster.py で投稿
4. 投稿済みが .env の POSTED_DATES に自動記録される
```

### 投稿文の書き方

- `x-post-template.md` のパターンを参照
- 140文字以内
- ハッシュタグ：#東かがわ #自家焙煎コーヒー #旅するコーヒー屋 から2〜3個
- 口調：手紙を書くような温かさ

### x-post-log.md への追記フォーマット

```markdown
## YYYY年M月D日
**テーマ：** テーマを一言で
**投稿文：**
ここに本文を書く

#ハッシュタグ

---
```

## Instagram 投稿フロー

1. 写真を選ぶ（豆・焙煎シーン・手紙・マルシェ風景）
2. キャプションを考える（X投稿文を参考に拡張可）
3. 手動投稿
4. `kpi.md` にいいね数・保存数を記録

---

## LINE公式 配信フロー

1. 配信テーマを決める（イベント告知 / 季節の一言 / お礼 など）
2. 友だち全員 or セグメント配信
3. `kpi.md` に開封率・クリック数を記録
