目的:

公開1週目で目標30万円の30%、つまり90,000円に到達する。

現状:

- 2026-05-02時点 支援総額: 55,000円
- 達成率: 18%
- 支援者数: 2人
- 1週目30%ライン: 90,000円
- 追加で必要な金額: 35,000円
- 期限目安: 2026-05-07

QRコード:

- 素材: `images/campfire/campfire-qrcode.svg`
- 用途: Instagramストーリーズ、LINE配信、マルシェ・対面告知、印刷物、活動報告画像

## 戦略意図

CAMPFIREは最初の1週間で勢いを見せることが重要。
30%到達は、見た人に「もう動いている」「応援が集まり始めている」と伝わりやすい。

ただし、煽りすぎない。Silver Treeの言葉は、押し売りではなく「一緒に旅の入口を作る」温度で伝える。

## QRコードの使い方

### Instagramストーリーズ

画像にQRを入れる場合:

- 1枚目: 現在の数字 55,000円 / 18% / 2人
- 2枚目: 1週目の目標 90,000円
- 3枚目: QRコード + 「ページを見るだけでも力になります」

文例:

```
2人目の支援が届きました。
55,000円、18%。

1週目で90,000円を目指しています。
ページを見るだけでも、本当に力になります。
```

### LINE

LINEではURLとQRの両方を使える。
既存顧客向けなので、数字よりも「近況報告」として送る。

文例:

```
CAMPFIRE、2人目の支援が届きました。
現在55,000円、18%です。

最初の1週間で90,000円まで届くと、
プロジェクトに勢いがつきます。

支援でなくても、ページを見る・誰かに送るだけでも力になります。
QRコードから見られます。
https://camp-fire.jp/projects/944948
```

### X

XではQRよりURL・プロフィール誘導を優先。
QR画像を使う場合は、実写や進捗画像と組み合わせる。

文例:

```
2人目の支援が届きました。
55,000円、18%。

1週目で90,000円まで届けたい。
あと35,000円です。

支援だけでなく、シェアも大きな力になります。
プロフィールのリンクから。
```

## 避けること

- 「あと少し」「お願いします」だけで連投しない。
- 目標未達の不安を前面に出さない。
- 支援者名を出さない。
- QRだけを貼って説明を省かない。
- 知識系投稿を未検証のまま画像化しない。
