CAMPFIREは最初の1週間で勢いを見せることが重要。

ただし、煽りすぎない。Silver Treeの言葉は、押し売りではなく「一緒に旅の入口を作る」温度で伝える。

## QRコードの使い方

### Instagramストーリーズ

画像にQRを入れる場合:

- 1枚目: 現在の数字 84,500円 / 28%
- 2枚目: 1週目の目標 90,000円
- 3枚目: QRコード + 「ページを見るだけでも力になります」

文例:

```
84,500円、28%まで来ました。
1週目30%の90,000円まで、あと5,500円です。

ページを見るだけでも、本当に力になります。
```

### LINE

LINEではURLとQRの両方を使える。
既存顧客向けなので、数字よりも「近況報告」として送る。

文例:

```
CAMPFIRE、84,500円、28%まで来ました。
1週目30%の90,000円まで、あと5,500円です。

最初の1週間で90,000円まで届くと、
プロジェクトに勢いがつきます。

支援でなくても、ページを見る・誰かに送るだけでも力になります。
QRコードから見られます。
https://camp-fire.jp/projects/944948
```

### X

XではQRよりURL・プロフィール誘導を優先。
QR画像を使う場合は、実写や進捗画像と組み合わせる。

文例:

```
84,500円、28%まで来ました。

1週目で90,000円まで届けたい。
あと5,500円です。

支援だけでなく、シェアも大きな力になります。
プロフィールのリンクから。
```

## 避けること

- 「あと少し」「お願いします」だけで連投しない。
- 目標未達の不安を前面に出さない。
- 支援者名を出さない。
- QRだけを貼って説明を省かない。
- 知識系投稿を未検証のまま画像化しない。
