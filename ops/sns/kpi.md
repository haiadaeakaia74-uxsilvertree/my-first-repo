# Silver Tree Coffee Roaster — X KPI トラッキング

## 実験概要

**目的**: 自動投稿（#ひとときの4分）＋画像付き投稿が、エンゲージメントと購買につながるかを検証  
**期間**: Phase 1（ベースライン）→ Phase 2（画像あり自動投稿）→ Phase 3（購買転換）  
**判断軸**: インプレッション・いいね率・プロフィールアクセス・LINE登録・購入

## Phase 3 購買転換の検証（4/18〜）

### 追跡すべき購買導線

```
X投稿
  → プロフィールアクセス
    → LINEリンク / ショップリンク
      → LINE登録 or ショップ閲覧
        → 購入
```

### 購買につながる仮説（検証中）

| 仮説 | 根拠 |
|------|------|
| 木曜夜シリーズ → LINE登録 | いいね率最高、感情的な共鳴 |
| 商品ストーリー投稿 → 購入 | Dark Blend(307imp)はブランド背景を語っている |
| 画像付き → プロフィール流入 | 視覚的な認知が購買意欲に先行 |
| リアル体験投稿 → 信頼形成 | コーヒーフェス等のリアリティが信頼につながる |
