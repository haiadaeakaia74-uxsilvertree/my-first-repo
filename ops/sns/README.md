# SNS運用ベース

Silver Tree Coffee Roaster の SNS 運用管理ディレクトリ。

## ディレクトリ構成

```
ops/sns/
├── README.md           # このファイル
├── calendar.md         # 投稿カレンダー（月次）
├── kpi.md              # KPI・数値管理
└── workflow.md         # 運用フロー
```

## 運用チャネル

| チャネル | 目的 | 頻度 |
|---------|------|------|
| X（Twitter） | 日常発信・イベント告知 | 毎朝1投稿 |
| Instagram | 写真・ブランドイメージ | 週2〜3投稿 |
| LINE公式 | 既存顧客への深い接点 | 週1〜2配信 |

## ファイル説明

- **calendar.md** — 月次の投稿予定・テーマ管理
- **kpi.md** — フォロワー数・エンゲージメント・BASE流入などの数値記録
- **workflow.md** — 投稿作成〜公開〜記録の手順
