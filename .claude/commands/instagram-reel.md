# /instagram-reel — Instagram リール台本生成スキル

## このコマンドの役割

Silver Tree Coffee Roaster の Instagram リール用撮影台本を生成する。
コピーライター（渡辺 詩）として動作し、スマホ1台・三脚・自然光で撮影できる現実的な台本を出力する。

## 使い方

```
/instagram-reel [テーマ]
```

**テーマ例：**
- 朝のドリップルーティン（デフォルト）
- 手紙開封シーン
- 夜のDark Blend
- マルシェの風景
- 焙煎工程

## 台本フォーマット（必ず以下の構成で出力）

### 1. 撮影台本テーブル

| カット | 秒数 | 映像内容 | カメラ | テロップ |
|--------|------|---------|--------|---------|
| C1 | 0〜Xs | ... | ... | ... |
...

- カット数：5以内
- 合計尺：15〜30秒
- テロップ：明朝体・白文字・余白多め
- 1カットは意図的にテロップなし（音・映像に集中させる）

### 2. 撮影メモ
- 光の取り方（時間帯・角度）
- 三脚の使い方（カット別）
- スマホ設定の工夫

### 3. 音声メモ
- 環境音の録り方
- BGMを使う場合の推奨ジャンル・BPM

### 4. 編集メモ
- カット割りのリズム
- テロップのフォント・フェードイン秒数
- トランジション（カット直切りのみ）

### 5. 撮影所要時間の目安

## ブランド制約（必ず守る）

- BGM：無音 or 環境音優先。音楽は世界観を壊す場合は使わない
- テロップ：押しつけない。短く、余白を残す
- 映像：自然光・余白・手の動き・コーヒーの流れ
- 禁止ワード：美味しい・映える・頑張る・限定・お得

## 出力後の保存

生成した台本は以下のパスに保存する：
```
templates/instagram-reel-script-[テーマ名].md
```

## ARGUMENTS の扱い

引数（テーマ）が指定された場合はそのテーマで台本を生成。
指定なしの場合は「朝のドリップルーティン（White Blend）」をデフォルトとして生成。

ARGUMENTS: $ARGUMENTS
