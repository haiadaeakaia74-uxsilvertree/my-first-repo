# /finance — 財務部門スキル

## このコマンドの役割

売上・コスト・価格・ROIに関するタスクを財務マネージャーに委任する。

## ルーティング詳細

| タスク種別 | 担当エージェント | 参照ファイル |
|-----------|----------------|------------|
| 月次売上レポート | 財務マネージャー | `agents/finance/finance-manager.md` |
| イベント採算計算 | 財務マネージャー | `guidelines/07-financial-management.md` |
| 商品価格設定 | 財務マネージャー + 商品設計士 | 財務 + 商品 |
| 顧客LTV分析 | 財務マネージャー | `guidelines/01-company-overview.md` |
| 施策ROI試算 | 財務マネージャー | — |
| 資金繰り確認 | 財務マネージャー | `guidelines/07-financial-management.md` |

## 起動プロンプト

```
あなたは Silver Tree Coffee Roaster の財務マネージャーです。
`agents/finance/finance-manager.md` の定義に従って動作してください。
`guidelines/07-financial-management.md` と `guidelines/01-company-overview.md` を参照し、
以下のタスクに取り組んでください：

[タスク内容]
```

## 典型的なリクエスト例

- 「今月の売上をまとめてほしい」
- 「このマルシェに出ると採算は合う？」
- 「コーヒー豆の価格を上げていい？」
- 「年間目標を設定したい」
- 「広瀬くんが来なくなったら売上はどうなる？」
- 「利益率を上げるにはどうすればいい？」

## 財務データの管理

財務マネージャーは `guidelines/07-financial-management.md` に定義された：
- ベースライン数値（年間198,000円売上）を参照
- 顧客別売上データを参照
- チャネル別売上比率を参照

## 出力テンプレート

財務レポートは以下の形式：
- 数字は必ず円単位で明記
- 前年・前月比較を含める
- グラフ相当の情報をテキスト表現で出力
- 必ずリスクと機会の両方を記載
