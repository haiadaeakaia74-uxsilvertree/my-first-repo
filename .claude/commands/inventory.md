# /inventory — 生豆在庫グラフ更新スキル

## このコマンドの役割

生豆の在庫データを受け取り、`reports/inventory/stock-chart.html` を更新してコミット・プッシュする。

## 使い方

以下の形式でデータを伝えてください：

```
銘柄名 仕入れ量
使用済み量

例：
ブラジル 5kg、300g使用
エチオピア 5kg、2.5kg使用
コロンビア 5kg、1kg使用
コスタリカ 5kg、2kg使用
```

## 実行手順

1. 入力データを解析して残量・消費率を計算する
2. `reports/inventory/stock-chart.html` のデータ部分を更新する（HTMLテンプレートは維持）
3. 集計日を今日の日付に更新する
4. git commit & push する

## HTMLテンプレートのデータ更新箇所

以下の値を書き換える：

- `.item-header` の銘柄名・残量・使用量テキスト
- `.seg-remain` の `width` パーセント と kg表示
- `.seg-used` の `width` パーセント と kg表示
- `<td>` テーブルの各行数値
- `⚠️` 警告は消費率50%以上の銘柄に付ける（`.warn` クラスも付与）
- `.date` の集計日

## 消費率の警告基準

| 消費率 | 表示 |
|--------|------|
| 〜49% | 通常（緑） |
| 50%〜 | 警告（赤 + ⚠️） |

## 完了後

「グラフを更新しました。同じURLで再読み込みしてください。」と報告する。
