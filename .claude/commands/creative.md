# /creative — クリエイティブ部門スキル

## このコマンドの役割

手紙・ブランド表現・世界観管理に関するタスクをクリエイティブ部門に委任する。

## ルーティング詳細

| タスク種別 | 担当エージェント | 参照ファイル |
|-----------|----------------|------------|
| 商品同梱の手紙作成 | レターライター | `agents/creative/letter-writer.md` |
| 顧客個別の手紙 | レターライター（顧客サクセスと連携） | 顧客情報を先に取得 |
| 季節の便り | レターライター | `guidelines/02-brand-identity.md` |
| ブランドの一貫性評価 | ブランドディレクター | `agents/creative/brand-director.md` |
| 新デザイン要素の評価 | ブランドディレクター | `guidelines/02-brand-identity.md` |
| パッケージコンセプト | ブランドディレクター + 商品設計士 | クリエイティブ + 商品 |
| コピーのブランド評価 | ブランドディレクター | コピーライターの成果物を評価 |

## 起動プロンプト（レターライター）

```
あなたは Silver Tree Coffee Roaster のレターライターです。
`agents/creative/letter-writer.md` の定義に従って動作してください。
`guidelines/02-brand-identity.md` と `guidelines/04-customer-communication.md` を参照し、
以下の手紙を作成してください：

[手紙の目的・対象・シーン]
```

## 起動プロンプト（ブランドディレクター）

```
あなたは Silver Tree Coffee Roaster のブランドディレクターです。
`agents/creative/brand-director.md` の定義に従って動作してください。
`guidelines/02-brand-identity.md` を参照し、
以下のクリエイティブのブランド評価を行ってください：

[評価対象・内容]
```

## 典型的なリクエスト例

- 「商品に同梱する手紙を書いてほしい」
- 「春の手紙を作りたい」
- 「この文章、Silver Treeらしいかチェックしてほしい」
- 「ブランドカラーを整理したい」
- 「顧客○○さんへの手紙を書いてほしい」
- 「InstagramとXで世界観がバラバラになっている、統一したい」

## 生成と評価の分離（クリエイティブ）

手紙・コピーを制作する場合：
1. レターライター / コピーライターが生成
2. ブランドディレクターがブランド適合性を評価（別エージェントとして起動）
3. 評価結果を受けて修正
4. 社長が最終承認

## 出力テンプレート

- 手紙：`templates/customer-letter.md`
- ブランド評価レポート：独自形式（適合スコア付き）
