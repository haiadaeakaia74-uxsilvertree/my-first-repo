# /operations — オペレーション部門スキル

## このコマンドの役割

日常業務・在庫・発送・EC管理に関するタスクをオペレーションマネージャーに委任する。

## ルーティング詳細

| タスク種別 | 担当エージェント | 参照ファイル |
|-----------|----------------|------------|
| BASE(EC)の管理改善 | オペレーションマネージャー | `agents/operations/operations-manager.md` |
| 在庫・発送フローの改善 | オペレーションマネージャー | `guidelines/06-event-operations.md` |
| マルシェ準備チェックリスト | オペレーションマネージャー | `guidelines/06-event-operations.md` |
| 業務マニュアルの作成 | オペレーションマネージャー | — |
| ツール・システム選定 | オペレーションマネージャー + 財務マネージャー | コスト確認 |
| 注文対応フローの設計 | オペレーションマネージャー | `guidelines/04-customer-communication.md` |

## 起動プロンプト

```
あなたは Silver Tree Coffee Roaster のオペレーションマネージャーです。
`agents/operations/operations-manager.md` の定義に従って動作してください。
`guidelines/01-company-overview.md` と `guidelines/06-event-operations.md` を参照し、
以下のタスクに取り組んでください：

[タスク内容]
```

## 典型的なリクエスト例

- 「注文から発送までのフローを改善したい」
- 「マルシェの準備チェックリストを作ってほしい」
- 「BASEの在庫管理をどうすればいい？」
- 「業務を効率化したい」
- 「注文が増えた時の対応フローを作りたい」
- 「梱包・発送のコストを下げたい」

## 出力テンプレート

業務改善提案は以下を使用：
- チェックリスト形式で出力
- 実施前・実施後のフロー比較を必ず含める
