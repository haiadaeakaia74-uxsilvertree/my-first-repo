# /marche-prep — マルシェ出店準備スキル

Silver Tree Coffee Roaster のマルシェ・イベント出店に必要な準備をすべて整理する。
`guidelines/06-event-operations.md` を参照すること。

---

## Step 1：出店情報の確認

`$ARGUMENTS` から以下を読み取る。なければ質問する。

| 項目 | 必須 | 例 |
|------|------|-----|
| 出店日時 | ○ | 4月20日（日）10:00〜15:00 |
| 場所 | ○ | 引田・○○神社境内 |
| 規模 | △ | 小（1〜2人）/ 中（3〜5人）/ 大（5人以上） |
| 販売内容 | △ | ドリップ提供 / 豆販売 / 両方 |
| 移動手段 | △ | 軽バン / 自家用車 |

---

## Step 2：3つのリストを生成

### ① 事前準備チェックリスト（出店1週間前〜前日）

**1週間前**
- [ ] 出店場所の最終確認（場所・時間・料金）
- [ ] 天気予報チェック（雨対策の要否判断）
- [ ] 商品の在庫確認・不足分の手配
- [ ] 手紙・おみくじの枚数確認（最低30枚）
- [ ] SNS告知投稿（Instagram + X）

**3日前**
- [ ] 焙煎・豆の準備（出店分）
- [ ] 釣り銭の準備（1,000円×10枚・100円×30枚・10円×30枚）
- [ ] 値段表・POP類の確認
- [ ] QRコード（LINE・BASE）の印刷確認

**前日**
- [ ] 積み込みチェック（下記「持ち物リスト」参照）
- [ ] 充電確認（スマホ・モバイルバッテリー・電源）
- [ ] 氷・保冷材の準備（アイスコーヒー提供の場合）
- [ ] 早起きセット（目覚ましx2）

---

### ② 持ち物リスト

**什器・設備**
- [ ] テーブル（×__）
- [ ] 椅子（×__）
- [ ] テント / タープ
- [ ] ウェイト（風対策）
- [ ] 電源（延長コード・バッテリー）

**コーヒー器具**
- [ ] ドリッパー + フィルター
- [ ] ケトル（電気 or ガス）
- [ ] ガスコンロ + カセットガス
- [ ] スケール
- [ ] 温度計
- [ ] カップ（紙 or リユース）
- [ ] 豆（販売用 + 提供用）
- [ ] グラインダー

**販売・決済**
- [ ] 釣り銭セット
- [ ] Square / PayPay 決済端末
- [ ] 領収書・袋
- [ ] 値段表・POP
- [ ] 名刺 / QRカード

**Silver Tree 特有**
- [ ] おみくじ手紙（__枚）
- [ ] シーリングワックス + スタンプ
- [ ] 封筒
- [ ] 手提げ袋

**衛生・雑備**
- [ ] ウェットティッシュ / アルコール
- [ ] ゴミ袋（大・小）
- [ ] ガムテープ・ハサミ
- [ ] 雨対策（レインコート・防水カバー）

---

### ③ 当日タイムライン

出店時間に合わせて逆算して生成する。

```
[出店時間-2時間30分] 出発
[出店時間-2時間]     現地到着・駐車
[出店時間-1時間30分] テント・テーブル設営
[出店時間-45分]      器具セットアップ・試し抽出
[出店時間-15分]      値段表・手紙ディスプレイ
[出店時間]           オープン
[出店時間+中間]      SNSストーリーズ投稿（現場の雰囲気）
[終了30分前]         在庫・売上の確認
[終了時刻]           撤収開始
[撤収後]             売上記録・翌日SNS投稿のメモ
```

## Step 3：SNS告知文（おまけ）

出店告知用のInstagram + X 投稿文も生成する。

**Instagram用（キャプション）：**
```
今週末、[場所]に出ます。

[日時]

コーヒーと、手紙を持っていきます。

#[場所名] #香川コーヒー #旅するコーヒー屋 #silvertreecoffee
```

**X用（ツイート）：**
```
[日時]、[場所]に出店します。
White Blend / Dark Blend + おみくじ手紙。
[場所の特徴を一言]。
```
