# /broadcast-message — LINE/DM一括配信スキル

LINE公式・Instagram DMでの一斉配信・告知文を生成する。
「宣伝っぽく見える」「読まれない」「Silver Treeらしくない」を防ぐ。

`guidelines/04-customer-communication.md` と `guidelines/02-brand-identity.md` を必ず参照すること。

---

## Step 1：情報の確認

`$ARGUMENTS` から読み取る。なければ質問する。

| 項目 | 必須 | 例 |
|------|------|-----|
| 配信目的 | ○ | CF公開告知 / マルシェ出店案内 / 新商品告知 / お礼 |
| 対象 | ○ | LINE全員（現在 __名）/ 未購買者 / マルシェ来場者 / 購入者 |
| 核心情報 | ○ | 何を、いつ、どこで（必要なら） |
| リンク | △ | CAMPFIREページ / BASEリンク / なし |
| 配信タイミング | △ | 今日中 / 5/1 10:00 / 出店前日 |

---

## Step 2：文章設計の原則

### Silver Tree の配信文における禁止事項

```
NG ワード：
- 「ぜひ」「よろしくお願いします！」「お楽しみに！」
- 「限定」「急いで」「今だけ」「お得」
- 感嘆符の多用（「！！！」）
- 「宣伝っぽい書き出し」（「お知らせがあります！」）

NG 構造：
- 冒頭から売り込み
- 長すぎる文（10行以上）
- 箇条書きで情報を羅列（メールのような構造）
```

### Silver Tree の配信文における推奨

```
OK ワード：
- 「ご報告です」「お知らせがあります」（落ち着いたトーン）
- 「もし気になったら」「気が向いたら」（押しつけない）
- 「いつかどこかで」「またいつでも」（余白）

OK 構造：
- 書き出しはブランドエッセンス or 近況から（1文）
- 核心情報（2〜3文）
- リンクは最後に1つだけ
- 全体 5〜8 行以内
```

---

## Step 3：配信文の生成（A案・B案）

目的に応じたテンプレートを使って生成する：

### 配信タイプ別テンプレート

**タイプ1：新しい取り組み・CF・プロジェクト告知**
```
[近況や背景の一言（1〜2文）]

[核心情報 — 何をやるか、いつ始まるか（2〜3文）]

もし気になったら、のぞいてみてください。

[リンク or 「プロフのリンクから」]
```

**タイプ2：マルシェ・出店案内（個人的なトーン）**
```
[季節や近況から書き出し（1文）]

[日時・場所（シンプルに2文）]

[会いに来てほしい気持ちを押しつけない形で（1文）]
```

**タイプ3：商品・EC の告知**
```
[商品をなぜ作ったか or 誰のために作ったか（1〜2文）]

[商品の核心（1〜2文）]

よかったら見てみてください。→ [リンク]
```

**タイプ4：お礼・フォローアップ**
```
[具体的な出来事への感謝（1文、大げさにならず）]

[それについて思ったこと（1〜2文）]

[余韻で締め — またいつでも、という感じ]
```

## Step 4：配信記録

生成後に以下のファイルへの記録を促す（社長が確認・送信後に記録）：

### 記録フォーマット（x-post-log.md に準じた形式）

```markdown
## LINE配信 — [日付] [配信タイミング]

**対象**：[対象者]（___名）
**目的**：[配信目的]
**配信文**：
---
[実際に送った文章]
---
**開封率**：（確認できれば）
**リンクタップ数**：（確認できれば）
**反応**：（返信・問い合わせなど）
```

記録先：新規ファイル `line-broadcast-log.md`（ない場合は作成する）

---

## 出力フォーマット

```
## LINE配信文 — [目的]

### A案（控えめ・余白多め）
---
[配信文]
---

### B案（少し情報量多め）
---
[配信文]
---

### 推奨案と理由
[A/Bどちらを推奨するか + 理由 1〜2文]

### 配信後の記録
送信後に「配信しました」と伝えてください。line-broadcast-log.md に記録します。
```
