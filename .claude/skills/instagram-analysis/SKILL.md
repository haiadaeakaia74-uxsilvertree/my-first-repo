# /instagram-analysis — Instagramインサイト分析スキル

Silver Tree Coffee Roaster のInstagramデータを読み解き、次のアクションにつなげる。
`guidelines/05-sns-operations.md` と `guidelines/02-brand-identity.md` を参照すること。

---

## Step 1：データ収集

以下のデータを受け取る（スクリーンショット or 数値の入力）。

### プロフィールインサイト
| 指標 | 取得方法 |
|------|---------|
| 閲覧数 | プロダッシュボード → インサイト |
| インタラクション数 | 同上 |
| 新規フォロワー数 | 同上 |
| 投稿数（期間中） | フィード数を数える |
| 分析期間 | インサイト上部に表示 |

### 投稿別データ（上位・下位各5件）
| 指標 | 取得方法 |
|------|---------|
| 各投稿の閲覧数 | フィードの数値 |
| 投稿タイプ | 写真 / 動画 / リール / カルーセル |
| 投稿内容 | テーマの分類 |

---

## Step 2：分析フレームワーク

### ① KPI対比
| 指標 | 実績 | 目標 | 達成率 |
|------|------|------|--------|
| 新規フォロワー | | 月15〜20名 | |
| フィード投稿数 | | 月8〜10本 | |
| エンゲージメント率 | | 4% | |
| リール投稿数 | | 月1〜2本 | |

**エンゲージメント率の計算：**
```
インタラクション ÷ 閲覧数 × 100 = エンゲージメント率（%）
```

### ② 投稿タイプ別パフォーマンス分類

| 閲覧数帯 | 分類 | アクション |
|---------|------|-----------|
| 400以上 | 強い素材 | 積極的に増やす |
| 200〜399 | 中間 | 形式改善で伸びる可能性 |
| 200未満 | 弱い | 形式変更 or 減らす |

### ③ コンテンツタイプ別傾向

Silver Tree の実績パターン：

| コンテンツタイプ | 傾向 | 優先度 |
|---------------|------|--------|
| マルシェ・イベント写真 | 高（500超） | 最優先 |
| 手紙・シーリングワックス | 高（400〜450） | 最優先 |
| 焙煎士作業・舞台裏 | 中高（350〜400） | 積極投稿 |
| 商品写真 | 中（200〜300） | 改善余地あり |
| テキスト系投稿 | 低（150〜200） | 画像化推奨 |
| ドリップ静止画 | 低（120〜150） | リール化推奨 |
