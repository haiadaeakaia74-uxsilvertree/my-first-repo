# /omikuji-letter — おみくじ手紙ジェネレーター

Silver Tree Coffee Roaster のおみくじ手紙・同梱手紙のテキストを生成する。
`guidelines/02-brand-identity.md` と `guidelines/04-customer-communication.md` を必ず参照すること。

## おみくじ手紙とは

Silver Tree のマルシェ・EC注文で提供する「引いた人へ届く、一枚の手紙」。
占いではなく「その日の気持ちに寄り添う言葉」を届けるもの。

---

## Step 1：パラメータ確認

`$ARGUMENTS` から以下を読み取る。なければ質問する。

| パラメータ | デフォルト | 例 |
|-----------|-----------|-----|
| テーマ | 自由（回復・余白・朝・季節） | 「春」「疲れた人へ」「朝の時間」 |
| 枚数 | 10枚 | 5・10・25・30 |
| 用途 | マルシェ用 | マルシェ / EC同梱 / イベント |

---

## Step 2：手紙テキスト生成

### 文体ルール

- 1通：100〜180字（短すぎず、長すぎず）
- 書き出し：体験・情景・問いかけのどれかから入る
- 結び：余韻で終わる（「〜ですね」「〜かもしれない」「〜でいい」）
- 敬語：丁寧だが距離を置かない（「です・ます」ベース）

### 使う言葉・使わない言葉

**使う：** 回復・余白・静けさ・香り・深呼吸・4分・ゆっくり・そっと・朝・夜・手紙・言葉

**使わない：** 頑張る・ファイト・美味しい・お得・急ぐ・限定・おめでとう（占いっぽい表現）

### テンプレート構造

```
[情景 or 問いかけ（1〜2文）]

[今日の言葉（1〜3文）]

[余韻で締める（1文）]

Silver Tree Coffee Roaster
```

---

## Step 3：出力フォーマット

枚数分を番号付きで出力する。各通は区切り線で分ける。

```
━━━━━━━━━━━━━━━━━
No.1

[手紙本文]

Silver Tree Coffee Roaster
━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━
No.2

[手紙本文]

Silver Tree Coffee Roaster
━━━━━━━━━━━━━━━━━
```

出力後に以下を確認する：
- [ ] 全通にブランドトーンが一貫しているか
- [ ] 似た表現が重複していないか（多様性があるか）
- [ ] 「疲れた人の隣に置けるか」で読んで違和感がないか

---

## テーマ別ヒント

**春：** 桜・新しさ・始まり・でも急がなくていい
**夏：** 暑さ・静かな朝・氷・立ち止まる
**秋：** 落ち着き・深呼吸・夕暮れ・振り返り
**冬：** 温かさ・こたつ・手のひら・静けさ
**疲れた人へ：** 今日くらい休んでいい・呼吸・置いておく
**朝：** 余白・スイッチ・今日が始まる前
**夜：** 一日の終わり・ねぎらい・明日はまた
