## X投稿の心を動かす基準

文章の先には必ず見えない読者がいる。投稿文は作業報告ではなく、読者にどう行動してもらい、その先にどんな価値を渡すかから逆算する。

投稿前に必ず確認する。

1. 誰が読むのか。
   - まだ飲んだことがない人
   - 応援するか迷っている人
   - いつか出店に来てくれる人
2. その人にどう動いてほしいのか。
   - 飲んでみたい
   - 応援したい
   - 出店で会いたい
   - プロフィールやCAMPFIREを見たい
3. 行動した先にどんな価値を提供できるのか。
   - 新鮮なコーヒーを飲める
   - 旅するコーヒー屋を一緒に育てられる
   - 応援した活動が、実際の一杯になって返ってくる
4. だから今何を準備しているのか。
   - 軽バン、露店設備、水まわり、焙煎、商品設計はすべて手段として書く。

避ける書き方:

- 水タンク、採寸、棚、消毒用品などの設備説明だけで終わる。
- 「〇〇しました」「〇〇が必要です」だけで、読者の行動や価値が見えない。
- ポエム調で気分だけを書く。

良い順番:

```text
応援してくれた人に、新鮮な一杯を届けたい。
そのために町へ行ける露店活動を作る。
だから軽バンや露店設備を整える。
```

## 注意

実際の X への投稿実行は `/x-post` を使用する（副作用のある操作のため分離）。

X投稿予定を編集した場合は、ローカルの `x-post-log.md` 修正だけで完了にしない。
自動投稿は GitHub の `main` を読むため、必要な投稿関連ファイルだけをコミット・pushし、`origin/main` に最新本文が入ったことを確認してから完了報告する。
