# /instagram-reel — Instagramリール作成スキル

Silver Tree Coffee Roaster のブランドトーンに沿ったInstagramリール動画の撮影から投稿までをガイドする。
必要に応じて `guidelines/02-brand-identity.md` と `guidelines/05-sns-operations.md` を参照すること。

## 素材タイプ（2種類 + その他）

| タイプ | 尺 | 難易度 | 特徴 |
|--------|-----|--------|------|
| 4分ドリップ動画 | 60〜90秒 | ★★☆ | コンセプト直結・閲覧数・保存数が伸びやすい |
| シーリングワックスを押す瞬間 | 15〜30秒 | ★☆☆ | 短く撮りやすい・映え鉄板素材 |

---

## Step 1：素材タイプの確認

`$ARGUMENTS` にタイプ指定があればそのまま進む。
指定がない場合は以下を質問する：

```
どちらのリールから作りますか？

1. 4分ドリップ動画
   （コーヒーを淹れる工程を縦動画に。Silver Treeのコンセプトそのもの）

2. シーリングワックスを押す瞬間
   （手元アップの短尺動画。撮りやすく反応が取れやすい）
```

## Step 2：撮影ガイド

### ▼ 4分ドリップ動画

#### セットアップ（準備5分）

**機材**
- スマホ縦向き（9:16）
- 三脚 or 安定した台に固定（ブレ厳禁）
- アングルは「真上」か「斜め45度」どちらか1つに統一する

**光**
- 窓際の自然光のみ
- 晴れた午前中が最適
- 人工照明は使わない（ブランドビジュアルNG）

**小道具**
- ドリッパー・ケトル・スケール・カップ（必須）
- 手紙 or 封筒を1枚、隅に置く（Silver Treeらしさを加える）
- コーヒー豆を少量、自然に散らす

#### 撮影の流れ（全体4〜5分録画）

```
① 豆をグラインダーに入れる・挽く音    → 約10秒
② ドリッパーにフィルターをセット      → 約5秒
③ お湯を注ぎ始め（蒸らし）          → 約15秒  ★粉が膨らむ瞬間は必ず入れる
④ ゆっくり注ぐ・立ち上がる湯気       → 約20秒
⑤ カップに落ちるコーヒー            → 約10秒
⑥ カップを両手で持ち、ひと呼吸       → 約5秒  ← ここで終わる
```

> ③の蒸らしで粉がぷくっと膨らむ瞬間が一番再生される。必ず撮り逃さないこと。

#### 編集

- **尺：** 60〜90秒に収める
- **音：** 音楽なし・コーヒーの音（お湯・注ぐ音）だけ
- **カット：** 必要最低限。繋ぎすぎない
- **冒頭テキスト：** 最初の2〜3秒に1行だけ重ねる

```
「4分だけ、止まる。」
```

#### キャプション

```
コーヒーを淹れる4分間。
何もしなくていい。
ただ、香りと音を聞いていればいい。

それだけで、少し呼吸が変わる。

#スペシャルティコーヒー #珈琲 #手紙 #香川コーヒー
#回復 #4分 #ゆっくりコーヒー #自家焙煎
#silvertreecoffee #旅するコーヒー屋
```

---

### ▼ シーリングワックスを押す瞬間

#### セットアップ

- スマホ縦向き（9:16）・手元アップ
- 三脚 or テーブルに置いて固定
- 自然光 or 温かみのある間接照明

#### 撮影の流れ（15〜30秒録画）

```
① ワックスを溶かして垂らす    → 5秒
② スタンプを押す             → 3秒  ★押す瞬間がハイライト
③ ゆっくりスタンプを外す      → 5秒
④ 完成したシールを見せる      → 3秒
```

#### 編集

- **尺：** 15〜30秒
- **音：** 無音 or 環境音（クラシック・自然音）
- **テキスト：** なしでもよい。入れる場合は1行だけ

```
「封を、する。」
```

#### キャプション

```
手紙を送るとき、最後にワックスシールで封をする。

ただそれだけのことが、少しだけ特別な時間になる。

#手紙 #シーリングワックス #珈琲 #スペシャルティコーヒー
#香川コーヒー #自家焙煎 #silvertreecoffee #旅するコーヒー屋
#手紙のある生活 #コーヒーと手紙
```

## Step 3：投稿前チェックリスト

投稿前に以下を確認する：

- [ ] キャプションの1行目が「体験の描写」になっているか（説明・宣伝NG）
- [ ] 「美味しい・お得・限定・急ぐ」が入っていないか
- [ ] ハッシュタグが8〜12個あるか
- [ ] カバー画像（サムネイル）が美しく切り取れているか
- [ ] 音量・音質に問題がないか（静かな環境で確認）

---

## ブランドビジュアル原則（リール共通）

- **色：** オフホワイト・ベージュ・ブラウン・深緑の範囲
- **光：** 自然光・温かみのある光
- **空気感：** 余白を残す・詰め込まない
- **NG：** 賑やか・多色・人工的な明るさ・過度な加工

---

## 閲覧数を伸ばすポイント

1. **最初の1〜2秒で引き込む** — 動きのある場面（蒸らし・ワックスを垂らす）から始める
2. **音だけで伝わる動画を作る** — 無音で見ても美しいかを確認する
3. **保存される動画を意識する** — 「撮り方の参考にしたい」と思わせるクオリティ
4. **ストーリーズで予告する** — 投稿前日にストーリーズで「明日あげます」と告知する
