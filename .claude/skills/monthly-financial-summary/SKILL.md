# /monthly-financial-summary — 月次財務サマリースキル

毎月末の財務まとめを**構造化して記録**する。
手作業での集計・記録を自動化し、戦略判断に使えるレポートを生成する。

`guidelines/07-financial-management.md` を必ず参照すること。

---

## Step 1：売上データの収集

`$ARGUMENTS` から対象月・売上データを読み取る。なければ以下を質問する。

### 売上（チャネル別）
| チャネル | 記入欄 |
|---------|--------|
| マルシェ・イベント | 円（出店___回） |
| BASE（EC） | 円（___件） |
| 直接取引（廣瀬・川端・渡邊・池田さんほか） | 円 |
| その他 | 円 |

### コスト
| 項目 | 記入欄 |
|------|--------|
| 生豆仕入れ | 円 |
| 出店費（会場費・交通費） | 円 |
| 資材・包材 | 円 |
| その他経費 | 円 |

### 顧客・SNS数値（わかる範囲で）
| 項目 | 記入欄 |
|------|--------|
| X フォロワー数 | 名 |
| LINE 登録者数 | 名 |
| マルシェ新規 LINE 登録数 | 名 |
| BASE 新規購入者数 | 名 |

---

## Step 2：集計・分析

以下を自動計算する：

```
総売上 = マルシェ + BASE + 直接取引 + その他
総コスト = 生豆 + 出店費 + 資材 + その他
粗利益 = 総売上 - 総コスト
利益率 = 粗利益 ÷ 総売上 × 100

目標達成率 = 総売上 ÷ 月次目標（40,000円）× 100
前月比 = 今月総売上 ÷ 前月総売上 × 100

チャネル比率 = 各チャネル売上 ÷ 総売上 × 100
```

---

## Step 5：次月アクション提案

財務マネージャー 森 明として、数字に基づいた来月の提言を行う：

| 状況 | 提言 |
|------|------|
| 目標 100% 達成 | 来月はさらに上積みできる施策を検討 |
| 目標 80〜99% | どのチャネルが不足か特定し補強 |
| 目標 80% 未満 | 出店頻度の見直し or BASE施策の見直しを緊急検討 |
| BASE が 0〜1 件 | SNS → BASE の導線改善が最優先（`/strategy` に諮問） |
| 直接取引 80% 以上 | 依存リスクあり。新チャネル強化を提言 |
