# /roast-log — 焙煎記録スキル

Silver Tree Coffee Roaster の焙煎データを記録・整理・振り返りできる形にまとめる。

---

## Step 1：焙煎データの収集

`$ARGUMENTS` から豆名・日付を読み取る。以下の情報を確認する（わかる範囲でOK）。

### 基本情報
| 項目 | 例 |
|------|-----|
| 焙煎日 | 2026-04-17 |
| 豆の産地・名称 | エチオピア ナチュラル / White Blend 用 |
| 生豆重量 | 300g |
| 焙煎後重量 | 255g |
| 焙煎度 | 浅煎り / 中煎り / 深煎り |

### 工程データ（わかる範囲で）
| 項目 | 例 |
|------|-----|
| 投入温度 | 160℃ |
| 1ハゼ開始 | 9分30秒 / 185℃ |
| 2ハゼ開始 | 13分 / 205℃（深煎りの場合） |
| 排出温度 | 195℃ |
| 総焙煎時間 | 12分30秒 |

### 結果・評価
| 項目 | 例 |
|------|-----|
| 外観 | 色ムラなし / ハネ多め |
| 香り（生豆時） | フルーティー・蜂蜜 |
| 香り（焙煎後） | 甘さが増した・焦げ感なし |
| 次回への改善点 | 投入温度を5℃上げる |
| 総合評価 | ★★★★☆ |

---

## Step 2：記録フォーマットで出力

```markdown
# 焙煎ログ

**日付：** YYYY-MM-DD
**豆：** [産地・名称]
**用途：** [White Blend / Dark Blend / テスト]

---

## 数値データ

| 項目 | 値 |
|------|-----|
| 生豆重量 | g |
| 焙煎後重量 | g |
| 歩留まり | % （= 焙煎後/生豆 × 100）|
| 総焙煎時間 | 分 秒 |
| 投入温度 | ℃ |
| 1ハゼ | 分 秒 / ℃ |
| 排出温度 | ℃ |

---

## 工程メモ

[自由記述：気になったこと・気候・マシンの状態など]

---

## 評価

**香り：** [描写]
**外観：** [描写]
**総合：** ★___☆（5段階）

---

## 次回への改善点

- [ ] [改善点1]
- [ ] [改善点2]

---
*Silver Tree Coffee Roaster — 焙煎記録*
```

## 保存先

記録ファイルは以下の命名で保存することを推奨：

```
roast-logs/YYYY-MM-DD_[豆名].md
```

例：`roast-logs/2026-04-17_ethiopia-natural.md`
