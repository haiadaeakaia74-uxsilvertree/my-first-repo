# /x-design-note-post — 試作設計図つきX投稿

軽バンDIY、露店準備、CAMPFIRE資金用途、道具配置、試作設計図をXで伝える時に使う。
目的は「雰囲気」ではなく、支援金で何を現実にしているかを見せること。

## 使う場面

- 軽バンをキッチンカーではなく、露店道具の収納車として説明する。
- 支援金で整えるものを具体的に見せる。
- DIY初挑戦の不安と、準備している内容を一緒に伝える。
- 投稿文だけでは分かりにくい構造・配置・導線を画像化する。

## 投稿文の型

```text
[今やっている具体作業]をしています。

[なぜその形にするのか。キッチンカーではなく露店式、などの判断理由]

[具体物を列挙するだけで終わらせず、「AはBだからCする」の形で配置理由を書く]

[本音 or 次に確認すること。わくわく、不安、初挑戦、現場で迷わないか等]
```

### 例

```text
軽バンの試作設計図を作っています。

キッチンカーみたいに車内で営業する形ではなく、露店道具をきちんと積むための車にしたい。

給水タンク、排水バケツ、器具、豆、衛生用品、テント、折りたたみ台。

現場で迷わず出せるかどうかを先に考えています。
```

### 文体ルール

- 「〇〇。〇〇。〇〇。」の短文羅列だけにしない。
- 「水タンクは重いから下に置く」のように、対象・理由・行動を一文でつなぐ。
- 感想だけで終わらせず、なぜその判断をしたかを入れる。
- 語尾が同じ文を3行以上続けない。
- 比較できる投稿は、Before / 仮設計 / 完成イメージの順に見せる。

## 画像の型

手描き設計図風にする。フラットなアイコン図だけで終わらせない。

- 紙に描いた試作設計図、木枠、ピン留め、コーヒー染みの質感。
- 軽バンの横向きイラスト。ドアを開け、車内に収納棚と道具を見せる。
- 「キッチンカー」ではなく「露店の道具箱」「露店収納車」と明記する。
- ラベルは大きく、矢印で指す。
- 右側に「支援金で整えるもの」を箇条書きで入れる。
- 同じ画像を連続投稿で使い回さない。
- 変化を伝えたい時は、単体写真より比較画像を優先する。
- 比較画像は「現在の写真」「仮設計図」「出来上がり仮設計図」を1枚にまとめると、支援後の変化が伝わりやすい。
- 実車写真だけでは理由が伝わらない時は、収納・設営・比較のどれかが見える画像に差し替える。

### 入れるラベル例

- 給水タンク
- 排水バケツ
- コーヒー器具
- 焙煎豆
- 衛生用品
- テント
- 折りたたみ台
- 収納棚

### 画像タイトル例

```text
軽バン露店収納 設計メモ
車内で営業するより、現場で迷わず出せる形へ
```

## 禁止

- ポエム系にしない。
- 「湯気」「夜」「整える」「香り」だけで終わらせない。
- 何を準備しているか分からない抽象文にしない。
- キッチンカーとして誤解される画像にしない。
- 支援依頼だけで終わらせない。
- 同じ設計図画像を翌日にもそのまま再掲しない。
- 実車写真だけを置いて「収納車が必要」と説明しない。画像側でも変化が分かるようにする。
