# Security Policy

## 秘密情報の扱い

このリポジトリには、APIキー、アクセストークン、顧客情報、決済情報、個人連絡先をコミットしない。

秘密情報は以下で管理する。

- ローカル作業: `.env`
- GitHub Actions: GitHub Secrets
- 共有用の見本: `.env.template`

## X自動投稿

X APIキーはGitHub Secretsだけに保存する。`x_poster.py` や `x-post-log.md` に認証情報を書かない。

投稿予定を追加する前に、以下を確認する。

- 投稿文に公開してはいけない情報がない。
- 画像ファイルが存在する。
- 知識系投稿は根拠確認済み。
- CAMPFIRE、BASE、LINEなどのリンクは公開してよいURLである。

## インフォグラフィック

コーヒー知識、温度、時間、比率、産地、健康・安全に関する内容は、画像化前に必ず信憑性を確認する。

未確認の数値や断定表現は使わない。

## 事故時

秘密情報をコミットした可能性がある場合は、すぐに以下を行う。

1. 該当キーを無効化・再発行する。
2. GitHub Secretsを更新する。
3. 公開済み投稿に誤情報があれば訂正投稿または削除判断を行う。
4. `ops/handoff/` に経緯を記録する。
