# ガイドライン 02 — ブランドアイデンティティ

> Silver Tree Coffee Roaster のブランドの核心定義です。
> すべてのコミュニケーション・制作物はこのガイドラインに準拠してください。

---

## ブランドの核心

### ミッション
**あなたのいる町へ、焙煎したてのコーヒーを届ける**

### ビジョン
**4分で回復のスイッチを作る**

### ブランドエッセンス（1文で表すとしたら）
> 「疲れた人の隣に、そっと置かれるもの」

### ポジショニング
- カフェでも、コーヒーショップでも、豆専門店でもない
- 「回復体験」を売っている
- 「旅するコーヒー屋」＝軽バンに焙煎機を積み、顧客のいる場所に行く
- 50歳・店を持たない・一人でやる。それが等身大のブランド

### 🔑 最重要差別化ポイント：手回し焙煎機
**他のロースターとの決定的な違い**

| 項目 | 他のロースター | Silver Tree |
|------|-------------|------------|
| 焙煎機 | 高価な電動・ガス機 | **手回し焙煎機（安価・アナログ）** |
| 移動 | 固定店舗 | **軽バンに積んで旅できる** |
| 一般的問題 | — | 手回し独特の煙臭さが出やすい |
| Silver Treeの解決 | — | **独自の焙煎プロファイル管理で克服** |
| 結果 | — | アナログで究極のコーヒーを提供 |

> 「高価な機械じゃなくていい。手で回すから、どこへでも行ける。煙臭さは、プロファイルで変えた。」
> これがSilver Treeのコアストーリー。CFの軸にもなる。

## ブランドパーソナリティ

Silver Tree を「人」として表現すると：

| 属性 | 内容 |
|------|------|
| 年齢感 | 50代・落ち着いた大人・等身大 |
| 性格 | 静かで誠実・言葉を選ぶ・押しつけない |
| 得意なこと | 聴くこと・空気を読むこと・場を整えること |
| 話し方 | ゆっくり・丁寧・無駄がない |
| いる場所 | 人が少ない静かな場所・早朝・夕暮れ |

## ブランドキーワード

### 使う言葉
- 回復・静けさ・余白・呼吸・間（ま）
- 香り・深呼吸・朝・夜
- 丁寧・ゆっくり・そっと
- 4分・スイッチ・切り替え
- 旅・軽バン・町・届ける・移動
- 言葉・手紙（ギフトセット文脈のみ）

### 使わない言葉
- 美味しい（体験描写で代替する）
- お得・特典・割引（価値を下げる）
- 急ぐ・限定（焦燥感を生む）
- バズる・映える・フォロワー増加（外向きすぎる）
- 頑張る・ファイト（疲れた人に刺さらない）

## ビジュアルアイデンティティ

### カラーパレット
| 色 | 用途 |
|----|------|
| オフホワイト #F8F4EE | ベース・余白の色 |
| クリームベージュ #D4C5A9 | 紙・封筒の質感 |
| ウォームブラウン #6B4226 | コーヒー・土・深み |
| ディープグリーン #2D4A3E | 自然・落ち着き・Silver Tree |
| インクブラック #1C1C1C | テキスト・主張 |

**禁止色：** 原色・蛍光色・明度の高い派手な色

### フォント方向性
- 日本語：明朝体 or 筆記体感のあるもの（ゴシック体より柔らかい印象）
- 英語：セリフ体 or 手書き風（装飾的でない）
- 数字：細めのウェイト・余白を活かす

### 写真・ビジュアルの方針
- 自然光・温かみのある光
- 余白を多く取る（モノをたくさん写さない）
- 人の手・コーヒーカップ・手紙・窓辺
- フィルター：彩度を少し落とす・フィルム感
- 避けるもの：賑やか・多色・ポップ・人工的な明るさ

## コミュニケーションのトーン

### 書き言葉の原則
1. **短く書く** — 3行で言えることを5行にしない
2. **余白を作る** — 読む人が想像できるスペースを残す
3. **説明しすぎない** — 体験を描写する、効能を説明しない
4. **感情に寄り添う** — 共感から入り、提案は最後
5. **押しつけない** — 〜してください より 〜はいかがですか、より さらに控えめに

### SNS投稿のトーン
- X（Twitter）：日常の気づき・短文・体験の断片
- Instagram：視覚体験・空気感・キャプションは短め
- LINE：個人的・丁寧・一対一の感覚

### 絶対にやらない表現
- 営業感の強い文章（〇〇が▲▲円！ご注文はこちら！）
- 感情を押しつける文章（これを飲めば元気になります）
- 誇張・大げさな表現（最高に美味しい・絶対おすすめ）
- 指示形（〜してください・〜しましょう）を多用する

## ブランドストーリー

50歳で、軽バンに焙煎機を積んで、あなたのいる町へ行くことにした。

Silver Tree は「旅するコーヒー屋」として、店を持たずに顧客のいる場所へ向かう。
コーヒーを売るのではなく、「4分だけ立ち止まる許可」を届けている。

香りは鼻から入って、記憶を揺らす。
疲れた人が、このコーヒーを淹れる4分間に、少しだけ呼吸を取り戻す。
それが Silver Tree が存在する理由。

ギフトセットには、おみくじ手紙が入っている。
封を開けるまで内容はわからない。その日だけのために書かれた言葉が届く。

## ブランド評価チェックリスト

制作物・投稿・提案をブランドに照らし合わせる際は以下を確認：

- [ ] 「疲れた人の隣に置けるか」の視点で見ているか
- [ ] 使ってはいけない言葉・表現を使っていないか
- [ ] 余白が十分にあるか（詰め込みすぎていないか）
- [ ] ブランドカラーの範囲内か
- [ ] 押しつけていないか（読む人を急かしていないか）
- [ ] 4分・香り・旅するコーヒー屋のコンセプトとつながっているか
- [ ] 手紙への言及はギフトセット文脈のみか（手紙屋サービスは廃止済み）
