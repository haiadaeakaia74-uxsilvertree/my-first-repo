# ガイドライン 04 — 顧客コミュニケーション

> 顧客とのすべてのコミュニケーションに適用される基本ルールです。
> 顧客サクセスマネージャー・レターライター・コピーライターが特に参照します。

---

## コミュニケーションの基本哲学

Silver Tree の顧客対応は「接客」ではなく「関係」。
一回一回の取引より、長く続く関係性を大切にする。
顧客は「お客様」である前に、「コーヒーと手紙が好きな人」。

**3つの原則：**
1. **個人として接する** — 名前を呼ぶ・好みを覚える・文脈を持つ
2. **先回りしない** — 勧誘や営業より、聴くことを優先する
3. **余白を持たせる** — 毎回返事を求めない・押しつけない

---

## チャネル別コミュニケーション方針

### LINE（主要チャネル）

**ポジション：** 既存顧客との個人的なつながり
**トーン：** 友達感覚に近い・丁寧だが堅くない・絵文字は控えめに
**頻度：** 月1〜2回の自然な接触（注文後のお礼・近況確認等）

メッセージの長さ：
- 通常：2〜4行（短すぎず長すぎず）
- お礼・確認：1〜2行でOK
- 特別な連絡（イベント告知等）：5〜8行まで

**やること：**
- 注文後3日以内に到着確認メッセージ
- 購入後30日ごろに「その後どうですか？」メッセージ
- イベント前の告知（出るよ、よければ来てね レベルの温度感）

**やらないこと：**
- 毎週連絡して購入を促す
- 既読スルーを責める
- セール・値引き情報を積極的に送る

---

### メール（正式連絡・BASEお客様）

**ポジション：** 丁寧な正式コミュニケーション
**トーン：** LINEより少し丁寧・でも固すぎない
**件名ルール：** 短く・内容が分かる・ブランド名を入れない（迷惑メール対策）

テンプレート基本構成：
```
・挨拶（短く）
・本題（何を伝えたいか）
・お礼 or 次のアクション
・締め
・署名（Silver Tree + 名前）
```

---

### 対面（マルシェ・イベント）

**ポジション：** Silver Tree のブランド体験の最前線
**トーン：** 自然体・温かい・説明は短く・質問を引き出す

対面の原則：
1. まず相手が見ているものを観察する（話しかけるタイミングを測る）
2. 声をかける時は圧をかけない（「よかったら試飲どうぞ」）
3. 商品の説明は聞かれてから（先に話しすぎない）
4. 買わなかった人にも丁寧に（次回の種まき）

---

## 顧客プロフィール管理

顧客一人ひとりについて以下を把握・更新する：

| 項目 | 内容 |
|------|------|
| 名前・呼び方 | 呼びやすい名前・関係性の距離感 |
| 好みのコーヒー | 好みのブレンド・焙煎度 |
| 購買パターン | 購入頻度・量・チャネル |
| ライフスタイル推測 | 職業・生活リズム（分かる範囲で） |
| 印象的なエピソード | 手紙の感想・会話の記憶 |
| 最終接触日 | いつ最後に連絡したか |

## 問い合わせ対応フロー

```
問い合わせ受信
  ↓
24時間以内に返信（受け取った旨を伝える）
  ↓
内容を確認・回答準備
  ↓
48時間以内に正式回答
  ↓
必要に応じてフォローアップ
```

### クレーム対応の基本

1. **まず謝罪・共感**（理由の説明より先）
2. **事実確認**（何が起きたか聞く）
3. **解決策の提示**（再送・返金等）
4. **再発防止の説明**（同じことが起きないように）
5. **感謝**（教えてくれたことへのお礼）

**絶対にやらないこと：**
- 言い訳を先にする
- 対応を1週間以上放置する
- 顧客の感情を否定する

## 顧客へのメッセージ例文集

### 注文後のお礼（LINE）
```
○○さん、ご注文ありがとうございます。
明日には焙煎して、xx日頃に届く予定です。
ゆっくり楽しんでもらえたら嬉しいです。
```

### 到着確認（LINE）
```
そろそろ届いた頃かなと思って。
コーヒー、どうでしたか？
気が向いたら教えてもらえると嬉しいです。
```

### しばらく注文がない時（LINE）
```
こんにちは、Silver Tree です。
最近いかがでしょうか。
新しいブレンドが仕上がったので、よかったらまた遊びに来てください。
（プレッシャーじゃなくて、ただ声をかけたくなって）
```

### マルシェ告知（LINE）
```
○月○日、□□でマルシェに出ます。
よかったらふらっと来てもらえると嬉しいです。
```

---

## 注意事項

- 顧客の個人情報（名前・購買履歴）は外部に出力しない
- 顧客の名前を他の顧客に話さない
- 顧客から聞いた個人的な話は、その顧客とのコミュニケーション以外に使わない
