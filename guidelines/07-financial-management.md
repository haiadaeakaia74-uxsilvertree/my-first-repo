## 価格設定の計算方法

### コストベース計算

```
原価 = 生豆仕入れコスト + 焙煎コスト（電気代・時間換算）+ 梱包コスト
最低価格 = 原価 ÷ (1 - 目標利益率)
```

**目標利益率：** コーヒー豆 70%以上 / ドリップバッグ 65%以上

### 価値ベース計算

```
1杯分の原価 ≒ 10〜30円
1杯の体験価値 ≒ カフェの1杯（500〜700円）
Silver Tree の位置 ≒ 家で飲むプレミアムコーヒー（250〜400円/杯）
100g（10〜15杯分）の価値 ≒ 2,500〜6,000円
適正価格 ≒ 価値の40〜50%（取りすぎない・安売りしない）
```

## 採算計算テンプレート

### マルシェ出店の採算計算

```
【収入】
期待売上 = 想定来場者数 × 購入率 × 平均客単価
            例: 500人 × 5% × 2,000円 = 50,000円

【コスト】
出店費: ○○円
交通費: ○○円
材料費（想定販売分の原価）: ○○円
その他: ○○円
合計コスト: ○○円

【結果】
期待利益 = 期待売上 - 合計コスト
採算ライン = コスト ÷ 平均客単価 ÷ 購入率 = 必要来場者数
```

### 新施策のROI計算

```
ROI = (施策による追加売上 - 施策コスト) ÷ 施策コスト × 100%

例: SNS広告5,000円使って売上が20,000円増えた場合
ROI = (20,000 - 5,000) ÷ 5,000 × 100% = 300%
```

## 重要な財務指標の見方

### 顧客生涯価値（LTV）の計算

```
LTV = 年間購買額 × 継続年数（推定）

例: 川端さん
LTV = 20,000円/年 × 5年（推定） = 100,000円

→ この顧客を維持するために使えるコスト（CAC）の目安 = LTVの20〜30%
   = 20,000〜30,000円まで投資できる
```

### 損益分岐点

```
固定費（月）≒ 通信費・ツール費用・出店費を含む月次固定コスト
変動費率 = 1 - 利益率（現状: 1 - 78% = 22%）

損益分岐売上 = 固定費 ÷ (1 - 変動費率) = 固定費 ÷ 利益率
```

## 財務管理のルール

### 記録すること
- 全収入（チャネル別・顧客別）
- 全支出（仕入れ・出店費・交通費・ツール費）
- 在庫量（生豆・梱包材）

### 確認するタイミング
- 月次：前月の売上・利益・コストを集計
- 四半期：チャネル別・顧客別の推移確認
- 年次：年間実績の振り返りと翌年目標設定

### 警戒ライン
- 月売上が10,000円を下回った場合 → 緊急確認
- 利益率が60%を下回った場合 → コスト構造の見直し
- 上位顧客の購買頻度が通常の50%以下になった場合 → フォローアップ

## 価格改定の判断基準

| 状況 | アクション |
|------|-----------|
| 原材料コストが10%以上上昇 | 価格改定を検討 |
| 競合相場が大きく変動 | 市場調査後に検討 |
| 顧客から「高い」と言われた | 価値提供の見直し（値下げではなく） |
| 顧客から「もっと払いたい」という声 | 値上げ or プレミアムライン追加を検討 |
