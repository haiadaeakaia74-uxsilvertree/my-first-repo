# ガイドライン 06 — イベント・マルシェ運用マニュアル

> マルシェ・イベント出店の準備から当日運営・アフターフォローまでの手順を定めます。
> 営業戦略家・オペレーションマネージャーが参照します。

---

## マルシェ出店の目的

1. **新規顧客との接点**（最重要）— SNSでは届かない層へのリーチ
2. **ブランド体験の提供** — Silver Tree の世界観を直接体験してもらう
3. **売上の獲得** — 単日売上の最大化
4. **口コミ・紹介の種まき** — 「また来たい」「誰かに教えたい」を作る

**過去実績：** スモフェス（2025/11/30）単日売上15,000円

---

## 出店判断基準

| 条件 | 基準 |
|------|------|
| 出店費 | 5,000円以下が理想（超える場合は採算計算必須） |
| 想定客層 | 30〜50代・女性・感度の高い層が理想 |
| 立地 | 香川県内 or 移動コストを含めて採算が合う範囲 |
| 開催規模 | 小〜中規模（大規模フェスより温度感のある場が向いている） |
| 季節・天候 | 雨天中止の屋外は要確認 |

### 採算計算の式
```
期待売上 = 想定来場者数 × 購入率（目安5〜10%）× 客単価（目安1,500〜2,500円）
コスト = 出店費 + 交通費 + 材料費（想定販売量の原価）
利益 = 期待売上 - コスト
目標：利益 10,000円以上 or ブランド認知目的として判断
```

## 当日の運営フロー

### 設営（開始1〜1.5時間前）
1. テーブル設置・テーブルクロス
2. 商品の陳列（見やすく・取りやすく）
3. POP・値札の配置
4. 試飲の準備
5. 釣り銭・決済端末の確認
6. 全体の確認（お客様目線で見る）

### 接客の流れ
```
近づいてきたお客様を観察（話しかけるタイミングを測る）
  ↓
「よかったら試飲どうぞ」（プレッシャーをかけない声がけ）
  ↓
興味を持ったお客様には自然に商品を説明
（4分で回復・コーヒー＋手紙のコンセプトを話す）
  ↓
購入 → 丁寧に袋に入れる・手紙を添える
  ↓
「LINE登録してもらえると次の情報をお届けできます」
  ↓
退場時も笑顔で見送り（買わなかった人も含めて）
```

### LINE登録の誘導トーク例
「このQRコードからLINEを登録してもらうと、新しいコーヒーが出た時や次のイベントのお知らせを送らせてもらっています。もしよければ。」
