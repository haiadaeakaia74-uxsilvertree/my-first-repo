# ガイドライン 05 — SNS運用マニュアル

> Silver Tree Coffee Roaster の SNS 運用に関する全ルールを定めます。
> SNS戦略家・コピーライターが参照します。

---

## SNS運用の目的（優先順位順）

1. **ブランド認知の形成** — Silver Tree の世界観を伝える
2. **信頼の蓄積** — 毎日投稿し続けることで「続けている人」として信頼される
3. **購買への導線** — SNS を見た人が自然に購入を検討できる流れを作る
4. **コミュニティ形成** — 回復を大切にする人たちとの接点を育てる

**現状課題：** 3番の「購買への導線」が機能していない。これを改善することが最優先ミッション。

---

## X（Twitter）運用ルール

### 基本方針
- 毎朝投稿（継続を最優先）
- フォロワーへの価値提供を優先（宣伝より共感・気づき）
- 週1〜2回はコーヒーや商品に絡めた投稿
- 月1〜2回は購買導線を含む投稿（押しつけにならない形で）

### 投稿の構成比率（目安）

| カテゴリ | 比率 | 例 |
|---------|------|-----|
| 日常・気づき・ことば | 50% | 「今朝の4分、窓から光が差してた」 |
| コーヒー・商品 | 30% | 「White Blendの浅煎りは、朝の最初の一杯に合う」 |
| 購買導線（CTA） | 15% | 「BASEで注文できます + URL」 |
| 告知・イベント | 5% | 「○月○日マルシェに出ます」 |

### 投稿の文字数・形式
- 理想：100〜180字（短いほど力がある）
- 最大：280字（超えない）
- 改行：読みやすさのために適切に入れる
- ハッシュタグ：0〜2個（多用しない）
- 絵文字：使わない or 1個まで（ブランドに合わない）

### CTA（行動喚起）の入れ方
**NG例：**「ご注文はこちら！→ [URL]」
**OK例：**「もし試してみたい方はBASEから。→ [URL]」
**さらに良い例：**「この味、BASEで出しています。」（URLはプロフィールに誘導）

### 投稿禁止事項
- 他ブランドの批判・比較
- 政治・宗教への言及
- 売上・収益を嘆くネガティブ投稿
- 「なんかすごい」「マジで最高」等の軽い言葉

## Instagram 運用ルール

> 詳細戦略：`reports/base-analysis/26-instagram-strategy.md` 参照

### 基本方針（更新：2026-04-16）
- **役割：** 焙煎士としての「見える化」プラットフォーム
- **フィード週3本：** Carousel（火）＋ 静止画（木）＋ Reels（土）
- **Stories：** 木曜に必ず Link Sticker（BASE リンク）を設定
- ビジュアルの世界観を統一（旧「回復・癒し」→ 新「焙煎士の専門性・リアル」）

### フォーマット別役割

| フォーマット | 柱 | 目的 |
|------------|-----|------|
| Carousel（スライド） | ①専門知識 | 保存率↑→ アルゴリズム評価 |
| Reels（短動画） | ②リアル実況 | 新規発見・リーチ拡大 |
| 静止画 | ③商品ガイド | ブランドの蓄積・BASE CTA |
| Stories | 随時 + 木曜購買 CTA | Link Sticker で BASE へ誘導 |

### ビジュアルルール
- 白・ベージュ・ブラウン・深緑の範囲でまとめる
- 彩度を少し落とした自然光の写真
- 人工的な照明・過度な加工は避ける
- 余白を活かす（ものを詰め込まない）

### キャプションの書き方
- 1〜2 行目：事実 or 気づき（興味を引く）
- 中段：補足・解説・体験（100〜250 字）
- 末尾（商品投稿のみ）：「プロフのリンクから BASE で購入できます」
- ハッシュタグ：5〜15 個（本文と分けて最後に）

### 固定ハッシュタグ（毎回必ず）
```
#自家焙煎珈琲 #スペシャルティコーヒー #焙煎士
#旅するコーヒー屋 #silvertreecoffeeroaster
```

### 内容別追加ハッシュタグ
```
柱①知識：#コーヒーの豆知識 #ハンドドリップ #コーヒー好きと繋がりたい
柱②リアル：#コーヒー屋 #マルシェ #軽バン #キッチンカー
柱③商品：#コーヒーギフト #珈琲 #香川コーヒー #おみくじ手紙
```

## LINE公式 運用ルール

### 基本方針
- 購入者・関係者向けのクローズドなコミュニティ
- 過度な配信はしない（月1〜2回が上限）
- 告知より「気にかけている」メッセージを大切に

### 配信タイミング
- マルシェ・イベント前1週間：告知配信
- 新商品リリース時：紹介配信
- 季節の変わり目：近況・季節商品の案内

### 配信文の書き方
- LINE は「個人からのメッセージ」感覚で書く
- 件名的な書き出しより、いきなり本題で
- 200字以内を目安
- URL は末尾に1つだけ

## SNSの効果測定

### 見る指標

| 指標 | 意味 | 目標 |
|------|------|------|
| エンゲージメント率 | いいね・RT・返信 / インプレッション | 2%以上 |
| プロフィールアクセス数 | 投稿からプロフを見た人数 | 前月比維持〜増加 |
| BASE流入数 | SNSからBASEへのアクセス | 月10件以上を目標 |
| フォロワー増加数 | 純増フォロワー | 月+10〜30人 |

### 見ない指標（惑わされない）
- インプレッション数だけ（中身が伴わないと意味がない）
- バズった回数（Silver Tree はバズを目指さない）

## SNSクライシス対応

**ネガティブコメントを受けた場合：**
1. 感情的に反応しない（最低1時間冷却期間）
2. 事実確認をする
3. 誠実に・短く・一度だけ返す
4. 炎上しそうなら社長に確認してから返信

**誤情報・誤解を招く投稿をした場合：**
1. 即座に削除 or 訂正投稿
2. 大きな問題の場合はお詫び投稿
