# 食品移動販売車 仕様概要書
（保健所事前相談・許可申請 添付用）
提出先：東讃保健所

## 申請者情報

| 項目 | 内容 |
|------|------|
| 屋号・事業者名 | Silver Tree Coffee Roaster |
| 住所 | 香川県 |
| 販売品目 | コーヒー（ドリップ・ハンドドリップ）※アルコール類なし |

## 車両情報

| 項目 | 内容 |
|------|------|
| 車種・型式 | 三菱 タウンボックス ハイルーフ（5ナンバー） |
| 車両登録区分 | 自家用（5ナンバー軽乗用）＊食品移動販売車として改造 |
| 荷室有効寸法（概算） | 幅 約1,600mm × 奥行 約1,600mm × 高さ 約1,350mm |
| 電源 | 発電機（HONDA EU18iT1JN 定格1,800W）屋外設置 |

## 内装・構造

| 項目 | 内容 |
|------|------|
| 運転席との仕切り | コンパネ（12mm厚）による完全仕切り設置 食品側は耐水加工 |
| 床材 | クッションフロア（耐水）貼付 継ぎ目コーキング処理 |
| 内壁 | アルミ複合板（白）または耐水塗装 調理エリア周辺を重点施工 |
| 作業台 | アルミテーブル（耐水シート敷設） |
| 照明 | LEDライト 調理エリア上部固定 |

## 給排水設備

| 設備 | 仕様・容量 |
|------|----------|
| 給水タンク | 食品用ポリタンク 40L（密閉型） |
| 排水タンク | ポリタンク 40L（給水と分離設置） |
| 洗い設備 | 足踏み式ポンプシンク（非接触式） |
| 洗浄用シンク | ステンレスシンク（小型）器具洗浄用 **レバー式または自動センサー蛇口**（捻るタイプ不可） |
| 配管 | 食品用シリコンホース（内径13mm）全接続部ホースバンド締結 |
| 排水処理 | 排水タンクへ回収 外部への垂れ流しなし |

## 調理・販売機器

| 機器 | 仕様 |
|------|------|
| 湯沸かし器 | 電気ケトル（1,000W / 0.8L）使用 発電機電源 |
| ドリッパー | ハリオ V60（ハンドドリップ） |
| 計量器 | タイマー付きコーヒースケール |
| 提供容器 | 使い捨て紙カップ（蓋付き） |
| 加熱調理 | なし（ドリップコーヒーのみ） |
| 食材の保管 | コーヒー豆・粉のみ 密閉容器保管 常温管理 |

## 衛生管理

| 項目 | 内容 |
|------|------|
| 手洗い方法 | 足踏みポンプにより流水手洗い 石鹸・ペーパータオル常備 |
| 器具洗浄 | 営業後に洗浄・乾燥 次回使用前に確認 |
| 廃水管理 | 排水タンクに全量回収 適切な場所で廃棄 |
| ゴミ処理 | 使用済み紙カップ等は持ち帰り処分 |
| 食品取扱者 | 食品衛生責任者 1名常駐 |

## 備考・確認事項

- 販売品目はハンドドリップコーヒーのみ。加熱調理・生食品の取扱いなし。
- 5ナンバー（自家用）での申請について、改造・構造要件を事前に確認済みの上で施工する。
- 給排水タンク容量（各40L）・シンク配置は保健所の指導に従い変更対応可能。
- DIY施工完了後、実地検査を申請予定。
- 本書記載内容に不備・変更指示がある場合は速やかに修正対応いたします。
