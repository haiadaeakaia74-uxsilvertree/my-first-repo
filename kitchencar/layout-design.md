# キッチンカー レイアウト設計図

車両：三菱 タウンボックス ハイルーフ
荷室寸法：幅 約1,600mm × 奥行 約1,600mm × 高さ 約1,350mm

---

## 上面図（フロア レイアウト）

```
← 幅 約1,600mm →

┌────────────────────────────┐
│▓▓▓▓▓ コンパネ仕切り ▓▓▓▓▓│ ← 運転席側（前方）
├───────────┬────────────────┤  ↑
│ 給水タンク │                │  |
│   40L     │   作業台       │  |
│           │  （アルミ）    │  奥行
├───────────┤                │  約
│ 排水タンク │                │  1,600mm
│   40L     ├────────────────┤  |
│           │ ステンレスシンク│  |
├───────────┤  （器具洗浄）  │  |
│  収 納    │                │  |
│ スペース   │ 足踏みポンプ   │  ↓
└───────────┴────────────────┘
      ↑           ↑
   左側面       後部ドア（接客口）

発電機：後部ドア横 屋外設置
```

---

## 側面図（高さ方向）

```
高さ 約1,350mm

┌────────────────────────────┐ ← 天井
│  LEDライト（調理エリア上部）│
│                            │
│  アルミ複合板（内壁・白）  │ 約1,350mm
│                            │
│  ─────作業台（約850mm）─── │
│  足踏みポンプシンク         │
└────────────────────────────┘ ← 床（クッションフロア）
```

---

## ゾーニング

| ゾーン | 場所 | 内容 |
|--------|------|------|
| 仕切りゾーン | 前方 | コンパネ12mm・耐水加工 |
| タンクゾーン | 左側 | 給水40L・排水40L・収納 |
| 作業ゾーン | 右側 | アルミ作業台・シンク・足踏みポンプ |
| 接客口 | 後部ドア | 商品提供・カップ渡し |
| 発電機 | 車外後部 | HONDA EU18iT1JN 屋外設置 |

---

## 配管フロー

```
給水タンク（40L）
       ↓
  シリコンホース（内径13mm）
       ↓
足踏みポンプシンク（手洗い）
ステンレスシンク（器具洗浄）
       ↓
  シリコンホース（内径13mm）
       ↓
排水タンク（40L）← 全量回収・外部垂れ流しなし
```

---

## 電気系統

```
発電機 HONDA EU18iT1JN（1,800W）
       ↓
  延長コード（車内へ引き込み）
       ↓
  ├── 電気ケトル 1,000W
  └── LEDライト  20W
       合計：1,020W（定格1,800W以内で安全）
```
