# キッチンカー DIY工程リスト

## 全体概要

| フェーズ | 内容 | 所要時間（目安）|
|---------|------|-----------|
| Phase 1 | 車内清掃・寸法測定 | 0.5日 |
| Phase 2 | 仕切り・床・内壁施工 | 1〜2日 |
| Phase 3 | 給排水設備設置 | 1日 |
| Phase 4 | 作業台・照明・機器設置 | 0.5日 |
| Phase 5 | 動作確認・仕上げ | 0.5日 |
| **合計** | | **約3〜5日** |

---

## Phase 1：車内清掃・寸法測定

- [ ] 荷室内部を完全からっぽにする
- [ ] 内寸実測（幅・奥行・高さ）
- [ ] コンパネ・クッションフロア・アルミ複合板の必要サイズを計算
- [ ] 購入資材リストをホームセンターで確認

---

## Phase 2：仕切り・床・内壁施工

### 2-1. 運転席との仕切り（コンパネ 12mm）
- [ ] コンパネを必要サイズにカット
- [ ] 耐水塗装（表面にニスまたは耐水ペイント）
- [ ] 車内に固定（ボルト・ぬきどめ・接着剤併用）
- [ ] 食品側の隙間部分をコーキング処理

### 2-2. 床材（クッションフロア 耐水）
- [ ] 床面のゴミ・凹凸を撤去し平滑にする
- [ ] クッションフロアをサイズ切り・貼付
- [ ] 継ぎ目をコーキング処理

### 2-3. 内壁（アルミ複合板）
- [ ] 調理エリア周辺側面にアルミ複合板を切り込み
- [ ] ネジ・接着剤で固定
- [ ] 継ぎ目・端部をコーキング処理

---

## Phase 3：給排水設備設置

### 3-1. タンク配置
- [ ] 給水タンク（40L）の固定位置を決める
- [ ] 排水タンク（40L）を給水と分離して固定
- [ ] 両タンクが走行時に紀れないよう固定を確認

### 3-2. 配管
- [ ] シリコンホース（内径13mm）を必要長さにカット
- [ ] 全接続部にホースバンドでしっかり締める
- [ ] 水漏れテスト（水を流して全接続部確認）

### 3-3. シンク設置
- [ ] 足踏みポンプシンクを作業台返りに固定
- [ ] ステンレスシンク（小型）を機材洗浄用に設置
- [ ] 排水パイプを排水タンクに接続

---

## Phase 4：作業台・照明・機器設置

### 4-1. 作業台
- [ ] アルミテーブルを安定する位置に固定
- [ ] 耐水シートを表面に敷設

### 4-2. 照明
- [ ] LEDライトを調理エリア上部に固定
- [ ] 発電機からの配線を整理

### 4-3. 機器配置確認
- [ ] 電気ケトルの設置場所を決める
- [ ] ハリオ V60・スケールの収納場所を決める
- [ ] コーヒー豆密閉容器の収納個数を決める

---

## Phase 5：動作確認・仕上げ

- [ ] 発電機起動・電気ケトル動作確認
- [ ] 給水ポンプ・水流れ確認
- [ ] 排水タンクへの回収確認（外部漏れなし）
- [ ] 車両走行時の機材固定確認（試走テスト）
- [ ] 全体の清掃・泡消毒
- [ ] 内部フォト撮影（保健所許可用）

---

## 完了後：保健所手続き

- [ ] 実地検査の申請（東讃保健所へ連絡）
- [ ] 検査時に仕様概要書を持参
- [ ] 許可取得後、初回出店日を確定
