# daily/ フォルダ — 毎日の写真置き場

## 命名ルール

```
YYYY-MM-DD.jpg
例: 2026-04-20.jpg
```

## 使い方

1. 毎朝6:00までにこのフォルダに写真をアップする
2. ファイル名を `YYYY-MM-DD.jpg` にする（その日の日付）
3. x-post-log.md の `**画像：** daily` と書いた投稿に自動で使われる

## 対応フォーマット

- `.jpg` / `.jpeg` / `.png`（この順で探す）

## 注意

- ファイルがない場合は画像なしでテキストのみ投稿される
- 1日1枚。複数アップしても最初に見つかったものが使われる
