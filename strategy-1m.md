## 【マーケティング部門】伊藤 蓮 × 渡辺 詩 — 購買導線の設計

### BASEで売れない本質（診断）

> 「共感で終わって、行動につながっていない」——これだけ。

3つの構造的問題：
1. 「知っている」と「買いに行く」の間に橋がない
2. LINEの一斉配信が「宣伝」に見えた（Silver Treeのトーンと不一致）
3. BASEページが「体験」ではなく「商品」を売っている

### ⚠️ マーケティングからの警告
> **「焦り」がブランドを壊す。**
> 「急ぐ・限定・お得」を使った瞬間にSilver Treeは機能しなくなる。

### X投稿「BASEへの扉」（すぐ使える3案）

**案①（夜・21〜23時）**
```
お湯を注ぎながら、
何も考えていない4分間があった。

仕事の話でも、誰かへの返事でもなく、
ただ、香りと時間だけがそこにあった。

そういう時間を、届けています。
→ Silver Tree / BASE
```

**案②（朝）**
```
今日、画面を閉じた瞬間がありましたか。

自分で作れる、唯一の「通知なし」の時間かもしれない。

Silver Treeの豆、BASEで取り扱っています。
```

**案③（写真が強いとき）**
```
疲れていいよ、と思いながら焙煎しています。

White Blend — 朝の深呼吸に。
Dark Blend — 夜のリセットに。

詳しくはBASEにて。
```
