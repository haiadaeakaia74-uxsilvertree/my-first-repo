## 4. BASEチャネル — Q1実績と4月以降の戦略転換

### Q1 BASE実績：0円（理由判明）

```
2026年1〜3月のBASE売上: 0円

【原因】
  手紙屋事業（梅竹松：梅便り・竹便り・松便り）を
  2026年1月〜3月にBASEで新展開した。
  → 購入件数: 0件（反応なし）

【判断】
  2026年4月：事業ピボットを決定
  ・手紙屋サービスを3月末で廃止（手紙はギフトセット同梱のみ継続）
  ・BASEをコーヒー豆・ドリップバッグ専門に再集中
  ・クラウドファンディング（CAMPFIRE）強化に並行移行
```

### 4月〜6月 BASE回復ロードマップ

| 月 | 目標売上 | 注文数イメージ | 施策 |
|----|---------|-------------|------|
| 4月（ピボット直後） | 4,500円 | 200g×2 + ドリップ×1 | 商品ページ整備・SNSリンク更新 |
| 5月（CF告知連動） | 7,500円 | 200g×3 + ドリップ×2 | CF予告とBASEへの誘導投稿 |
| **6月（目標月）** | **10,000円** | **200g×4 + ドリップ×3** | **CF公開 → BASE認知拡大** |

> ✅ 6月BASE月次10,000円が今期の重点マイルストーン。CF公開と同月なので認知拡大との相乗効果が期待できる。

### クラファン × BASE 相乗効果（試算）

```
CF支援者（目標300人）のうち10%がBASEにアクセス → 30人
30人 × 平均1,500円 = 45,000円の一時的ブースト
→ 6〜7月で1〜2ヶ月分の目標を一気にクリアできる可能性
```

## 5. 年間目標達成見通し（2026年）

### 現ペース試算

```
Q1確定売上: 72,400円（月平均 24,133円）
2025年全体平均: 15,042円/月

→ 現ペースで年換算: 24,133円 × 12ヶ月 = 289,600円
→ 目標400,000円まであと: 110,400円（27.6%の差）
→ 現ペース達成率: 72.4%
```

### 修正シナリオ：2課題（マルシェ10回 × BASE月1万円）達成時

| チャネル | 内訳 | 年間売上 |
|---------|------|---------|
| 廣瀬くん | 5,000円 × 12ヶ月 | 60,000円 |
| 川端さん | 6,000円 × 12ヶ月 | 72,000円 |
| 渡邊さん等 | 2,000円 × 12ヶ月 | 24,000円 |
| マルシェ10回 | 20,000円 × 10回 | 200,000円 |
| BASE | 4〜6月 平均5,000円 × 3ヶ月 + 7〜12月 10,000円 × 6ヶ月 | 75,000円 |
| **合計** | — | **431,000円** |

```
✅ 2課題を両方達成なら → 431,000円（目標400,000円を107.8%で達成）
⚠️ マルシェのみ達成（BASE 0円継続）→ 356,000円（目標の89%止まり）
❌ どちらも未達 → 280,000円前後（目標の70%）

= マルシェ10回 × BASE月1万円の両立が目標達成の必要条件
```

### CFあり・軽バン完成シナリオ（7月完成想定）

```
✅ CF成功・軽バン7月完成ならマルシェ月2回体制が可能
  → 8〜12月 40,000〜50,000円/月の水準を達成
```
