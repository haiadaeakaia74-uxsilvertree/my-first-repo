## BASEのバリエーション設定手順

1. 商品管理 →「商品を追加」
2. 商品名・価格・説明文を入力
3. 「バリエーション」をオン
4. バリエーション名：「グラインド」
5. 選択肢を追加：
   - `豆のまま`
   - `挽き豆（中挽き）`
6. 価格差なし（同額）
7. 在庫数を各バリエーションに設定

## 写真の撮り方（参考）

| ポイント | 内容 |
|---------|------|
| 背景 | 白または木目・クラフト紙 |
| 光 | 自然光（窓際）・影を作らない |
| 構成 | 袋のみ / 袋＋豆 / ドリップシーンの3パターンがあると理想 |
| 統一感 | 全商品で背景・光の向きを揃える |

> ブレンド2品は雰囲気で撮る（朝の窓辺 / 夜の卓上）
> シングルオリジンは産地カードや豆を見せる

## 出品後にやること

1. **SNS（X）で告知投稿**
   - 「BASEに新しく出品しました」ではなく
   - 「ブラジル ベラビスタ農園のイエローブルボンを焙煎しました。こういう豆です→」という専門家目線で

2. **LINE公式で既存客に通知**
   - 渡邊さん・川端さん・広瀬さんには個別LINE（BASEへ誘導しない）
   - LINE公式登録者には新商品ラインアップを案内

3. **スペシャルティ④決定次第**
   - ④を追加出品
   - ギフトセット（7種）を出品
