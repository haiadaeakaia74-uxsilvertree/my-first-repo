# SNS戦略 刷新案
**担当：伊藤 蓮（SNS戦略家）**
**作成日：2026-04-09**
**背景：2026年1月〜3月の手紙＋SNS戦略で購買ゼロ。戦略を根本から見直す。**

---

## ⚠️ 最重要前提（間違えないこと）

> **手紙屋は捨てていない。**
> 3ヶ月やって機能しなかったのは「前面に押し出すやり方」の問題。手紙そのものの価値は変わらない。

```
今やること（2026年）
  → コーヒー専門家として売上を作る
  → ギフトの「おみくじ手紙」で手紙の価値を体験させる（仕掛け）

3〜10年後（仮説）
  → ギフトを受け取った人が「手紙って凄い」と気づき、言葉で回復が浸透
  → 手紙ビジネスが独立した形になり、独占ビジネスになっている可能性
```

**おみくじ手紙の役割：**
売り込まない。ギフトに入れる。体験させる。それだけ。
言葉の力を「語る」のではなく「届ける」ことで浸透させる。

**今の最優先：売上を作ること。手紙はその中に静かに生きている。**

---

## 診断：なぜ3ヶ月で0だったか

| 問題 | 内容 |
|------|------|
| ターゲットのズレ | 「疲れた人」は感情的には共鳴するが、コーヒーを"今すぐ買う"動機が薄い |
| 購買意図の不在 | 共感投稿→いいね→終わり。購入の必要性を感じさせていない |
| 競合との差別化不足 | 「癒し系」のポジションは多くのアカウントが持っている |
| コーヒーの専門性が見えない | Silver Treeが「なぜここで買うべきか」が伝わっていない |

> **結論：「疲れた人を癒す」ポジションから「コーヒーを知る人・買う人のための焙煎士」にシフトする。**

## 新ターゲット（2軸）

### ターゲットA：「コーヒー好きだけどどこで買えばいい人」

```
プロフィール
  ├── スーパーのコーヒーには満足していない
  ├── スペシャルティコーヒーに興味はあるが、どこを選べばいいか分からない
  ├── 「自分に合うコーヒー」を探している
  └── 購買意図が高い（選び方さえ分かれば買う）

Silver Treeとの接点
  → 産地・焙煎度の解説投稿を見て「ここで買ってみよう」になる
  → 具体的な商品（White / Dark / Mandheling）が選びやすい状態が必要
```

### ターゲットB：「個人でコーヒー屋をやりたい人」

```
プロフィール
  ├── いつか小さなコーヒー屋・キッチンカーをやりたいと思っている
  ├── 焙煎・仕入れ・営業許可・出店の実態を知りたい
  ├── 「一人でやっている人」のリアルな情報を求めている
  └── LTV高（長期フォロワー → 教わる存在 → ファン購買）

Silver Treeとの接点
  → 「旅するコーヒー屋の実況」が学びになる
  → 共感と尊敬からファンになり、豆を買う
```

## 新コンテンツ3本柱

### 柱①：コーヒー専門知識（ターゲットA向け）

「どこで買えばいいか」に答えるコンテンツ。

**投稿テーマ例：**
```
・「スペシャルティコーヒーとは何か、3行で」
・「蒸らしを30秒やる理由」
・「マンデリンが好きな人はこういう人です」
・「中煎りと深煎り、何が違うか」
・「スーパーのコーヒーとの本当の違い」
・「コーヒー豆の鮮度はなぜ大事か」
・「Silver TreeのWhite Blendが朝に向いている理由」
```

**CTA（購買につなげる型）：**
```
最後に「→ BASEで取り扱っています」を自然に添える
「気になった人はプロフィールから」
```

---

### 柱②：コーヒー屋リアル実況（ターゲットB向け）

「一人でコーヒー屋をやるとはどういうことか」のリアルな記録。

**投稿テーマ例：**
```
・「今日の焙煎、失敗した話」
・「マルシェ出店の準備リスト全部見せます」
・「保健所に聞いてきた話（露店営業許可）」
・「軽バンをDIYしてコーヒー屋にする話」
・「一人でコーヒー屋をやって3年の売上公開」
・「個人でコーヒー屋を始めるなら最初に買うもの」
・「スペシャルティコーヒーの仕入れ先をどう探すか」
```

**効果：**
- 「この人みたいになりたい」からフォロワー増加
- 「この人のコーヒーを飲んでみたい」から購買

---

### 柱③：商品紹介＋選び方ガイド（両ターゲット向け）

押し売りではなく「選ぶ手助け」をする形で商品を見せる。

**投稿テーマ例：**
```
・「朝に飲むならWhite、夜に飲むならDark。理由はこれです」
・「マンデリンが好きな人のために焙煎した話」
・「スペシャルティ○○（産地）を仕入れた。こういう味です」
・「ドリップバッグ7種セット、それぞれの違いを解説します」
```

**頻度：週に1〜2本。全体の20〜30%が目安。**

## 移行について

旧コンテンツ（#ひとときの4分・疲れた人へ）は**即日停止しない。**
新コンテンツを週3本混ぜていき、2週間かけて比率を逆転させる。

ブランドの根幹（コーヒー＋手紙・香りと言葉）は変えない。
**語り口と向け先を変えるだけ。**

## 手紙ビジネスの長期ロードマップ（仮説）

| フェーズ | 時期 | 手紙の位置づけ |
|---------|------|--------------|
| 現在 | 2026年 | ギフトの「おみくじ手紙」として静かに体験させる |
| 浸透期 | 2027〜2028年 | 「手紙が届いた」体験が口コミで広がり始める |
| 確立期 | 2029〜2030年 | 手紙の価値が認知され、単独でも求められるようになる |
| 独占期 | 2031年〜 | 「Silver Treeの手紙」が唯一無二のブランドとして自立 |

> これは仮説。でも十分にあり得るシナリオ。
> だから今、おみくじ手紙を丁寧に作り続けることには意味がある。
> 売れなくても、届けることで種を蒔いている。
