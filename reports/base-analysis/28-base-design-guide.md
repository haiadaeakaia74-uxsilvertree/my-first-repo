## ① 商品写真 改善ガイド

### 基本方針

```
・自然光（窓際）で撮る。照明ライトは使わない
・余白を多く取る（商品は画面の40〜60%に収める）
・背景はオフホワイト or クラフト紙 or 木目の一択
・彩度を少し落とす（スマホの「ビビッド」設定は切る）
・全商品で背景・光の向きを統一する
```

---

### 撮影パターン（各商品3枚が理想）

#### パターンA：メイン写真（一覧に表示される）
```
・袋を立てて置く
・背景：オフホワイトの紙 or 木目のテーブル
・光：左または右の窓から自然光
・袋の正面を向ける（商品名が読めるように）
・余白：上下左右に広めに取る
```

#### パターンB：豆のアップ
```
・袋から少し豆を出して木の板やクラフト紙の上に広げる
・マクロ気味に寄って撮る
・産地ごとに豆の色・大きさの違いが伝わる
・光：柔らかい自然光
```

#### パターンC：シーン写真（世界観）
```
White Blend → 朝の窓辺。白いカップ。手が添えてある
Dark Blend  → 夜の卓上。暗めの光。落ち着いた雰囲気
シングルオリジン → 産地カード + 豆 + カップ
ギフトセット → 包装済みの箱 + 手紙がのぞいている
```

---

### スマホ撮影の設定

| 設定 | 内容 |
|------|------|
| フォーマット | 正方形（1:1）または 4:3 |
| HDR | オフ（自然な光を活かすため） |
| フィルター | なし（後から編集） |
| 編集アプリ | Lightroom Mobile（無料）推奨 |

**Lightroom Mobileの編集値（目安）：**
```
露光量　  +0.3〜0.5（少し明るく）
コントラスト  -10（柔らかく）
彩度　   -15〜-20（落ち着かせる）
色温度　  +200〜300（暖かみを出す）
シャープ  +20
```

---

### NG写真チェックリスト

- [ ] 蛍光灯の下で撮った（影が硬い・色が青白い）
- [ ] 背景が賑やか（テーブルに他の物が写り込んでいる）
- [ ] 商品が画面いっぱいに詰まっている（余白なし）
- [ ] ビビッド・鮮やかすぎる色調
- [ ] 複数枚で背景・光の方向が統一されていない

### 商品サムネイル画像の統一ルール

| 商品タイプ | 背景 | 雰囲気 |
|-----------|------|--------|
| White Blend | オフホワイト | 明るい・朝 |
| Dark Blend | クラフト紙（茶） | 落ち着いた・夜 |
| シングルオリジン全品 | 木目 | 豆が主役 |
| ギフトセット | ディープグリーン布 | 特別感・贈り物 |

> **重要：** 全商品でフォントサイズ・配置・余白のバランスを統一すること。
> バラバラだとショップ全体が安っぽく見える。

### ヘッダー画像の無料作成ツール
- **Canva**（canva.com）→ 「BASEヘッダー」テンプレートあり
  - Silver Tree ブランドカラーを登録して使う
  - フォントは明朝体 or セリフ体を選ぶ
