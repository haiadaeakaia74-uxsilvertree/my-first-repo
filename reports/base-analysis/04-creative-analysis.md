## 手紙の役割再定義

- 手紙は「同梱物」ではなく「もう一つの商品」
- 設計思想：「置いてくる感覚」
- 目標：「読まなくてもいい。でも読んでよかった」

---

## 封筒・デザイン・質感 推奨仕様

| 項目 | 推奨 |
|------|------|
| 素材 | クラフト紙 or コットンペーパー |
| サイズ | A6（個人的な手紙感） |
| カラー | クリームベージュ #D4C5A9 |
| 香り | 焙煎後すぐ封入（自然移し、コスト0） |
| 装飾 | 1点のみ（スタンプ or 封蝋） |
| 封筒の裏 | 「急がなくていいです」一行印刷 |

---

## 手紙の文章スタイルガイド

- 主語：「あなた」または「時間・場面」（Silver Tree を主語にしすぎない）
- 文字数：100〜200字（余白が文章の一部）
- 日時：書かない → 「春のはじまりに」等の季節の言葉
- 語尾：問いかけで終わらない。余韻で終わる
- 署名：汎用手紙は「Silver Tree」のみ

---

## 手紙文サンプル（White Blend・春）

```
　　　　　春のはじまりに


今日も、ここまで来ましたね。

コーヒーを淹れる時間は、4分ほどです。
豆を挽いて、お湯を沸かして、静かに注ぐ。
それだけのことが、どこかで呼吸を整えてくれます。

急ぐことは何もありません。
この一杯が終わるまで、今日はまだ始まらなくていい。

　　　　　　　　　　　Silver Tree
```

## BASEページ 改善ルール

**冒頭は情景描写から始める（スペックは後半）**

- White：「仕事に出る前の、誰もいない台所。窓から差し込む光の中でこれを淹れる。」
- Dark：「長い一日が終わって、ようやく座れた夜。深く息を吐くように、これを淹れる。」

**手紙の説明文を必ず追加：**
「コーヒーと一緒に、一枚の手紙が届きます。読んでも読まなくてもいい。ただそこにあります。」

**ブランド禁止ワード：**
「美味しい」「お得」「急ぐ」「頑張る」「豊かなコーヒーライフ」「贅沢」
