## Silver Tree 独自設計（律の提案）

### ①「4分待つ体験」設計
```
注文 → 「4分ほどいただきます。その間に1枚どうぞ」
     → 手紙おみくじを引いてもらう
     → ドリップ4分間 = 手紙を読む時間
     → コーヒー受け取り = 4分回復体験済み
```

### ②後部300mmに「手紙コーナー」設置
荷室奥行き1,600mmのうち後部300mmを手紙スペースとして確保。
- 手紙おみくじ ディスプレイ
- 便箋・封筒・ペンホルダー
- 「ここで書いていけます」のサイン

### ③コーヒー豆の「見せる保管」
- White Blend → 白ラベル 透明キャニスター
- Dark Blend  → 黒ラベル 透明キャニスター
- 本日の限定豆 → 手書きラベル 1種類

### ④香りを「外に漏らす」設計
リアドアを開けた瞬間にコーヒーの香りが広がるよう豆の配置を設計。
広告費ゼロの集客装置。

### ⑤#ひとときの4分 QRコード掲示
紙カップ蓋の内側または車体後部に貼付。
「この4分が良かったら #ひとときの4分 で投稿してください。」

## 1日の運営フロー

### 出動前
- 豆の確認（White・Dark・当日限定）
- 給水タンク満水確認
- 発電機燃料確認
- 手紙おみくじ補充確認
- 紙カップ・蓋の在庫確認

### 営業中
- リアドア開放 → 香りが広がる
- 接客：「4分ほどいただきます。その間に1枚どうぞ」
- ドリップ中に手紙を引いてもらう
- カップ渡し時：「また来てください」

### 撤収後
- 排水タンク廃棄（適切な場所）
- 器具洗浄・乾燥
- 豆の残量記録
- 売上記録（KPIダッシュボードに入力）
