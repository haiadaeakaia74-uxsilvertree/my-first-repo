**背景：SNS戦略刷新（25-sns-strategy-pivot.md）に合わせ Instagram 方針を全面更新**

---

## 診断：旧 Instagram の問題点

| 問題 | 内容 |
|------|------|
| コンテンツ方針のズレ | 旧 X 戦略（回復・詩的）と同期していた。新戦略（専門性・リアル）に未対応 |
| Reels 未活用 | 発見機能として最強のフォーマットを使っていない |
| Carousel なし | 教育コンテンツ（柱①）に最適なフォーマットを使っていない |
| CTA が弱い | BIO リンクのみ。Stories の Link Sticker を活用していない |
| 世界観の更新なし | 「回復・疲れた人」の世界観のまま。「焙煎士の専門性」に移行できていない |

## 新方針：Instagram の役割

```
X（声・言葉・知識）
  ↓ 気になった人が
Instagram（映像・プロセス・世界観の蓄積）
  ↓ ファンになった人が
BASE（購入）
```

**Instagram = 焙煎士としての「見える化」プラットフォーム**
- 豆が焙煎されるプロセス
- マルシェ出店の準備・空気感
- コーヒーの知識をビジュアルで分かりやすく

## フォーマット別戦略

### Reels（柱② リアル実況）— 発見エンジン

**目的：** フォロワー外への露出。新規流入の主力。

**内容例：**
```
・焙煎機の火が入る瞬間（15 秒）
・ハンドドリップを正面から撮った動画（30 秒）
・マルシェ出店の準備 → 完成 → 一杯目を淹れる（60 秒）
・軽バンのDIY過程（タイムラプス）
・焙煎中の豆の色変化（スローモーション）
```

**撮影・編集ルール：**
- BGM：著作権フリーのインスト（落ち着いたテンポ）
- テキストオーバーレイ：1〜2 行のみ。白または生成り色
- 最初の 1〜2 秒で「引き」を作る（豆が入る瞬間など）
- エンディング：Silver Tree ロゴ + 「プロフのリンクから」

**投稿タイミング：** 土曜または日曜の 9〜11 時

---

### Carousel（柱① 専門知識）— 保存率の高いコンテンツ

**目的：** 保存 → アルゴリズムに評価される → リーチ拡大。

**1 枚目のルール（最重要）:**
- タイトルは「問い」か「数字」で始める
  - ○「スペシャルティコーヒーって何が違うの？」
  - ○「焙煎度は3種類。あなたはどれ？」
  - ×「コーヒーの豆知識」（興味を引かない）

**スライド構成（5〜8 枚が最適）:**
```
1 枚目：タイトル（問い or 数字）
2〜6 枚目：解説（1 スライド 1 ポイント）
最終枚：まとめ + CTA（「BASEで購入 / プロフのリンクから」）
```

**Carousel ネタ 10 選（すぐ使える）：**
```
1. 「蒸らし 30 秒の科学」（なぜ必要か）
2. 「スペシャルティコーヒーとは、3 枚で分かる」
3. 「焙煎度 3 種類の違い（浅・中・深）」
4. 「マンデリンが好きな人の特徴」
5. 「朝のコーヒーに White Blend を選ぶ理由」
6. 「コーヒー豆の鮮度：焙煎後 2 週間以内に飲むべき理由」
7. 「ハンドドリップの道具、最低限これだけ」
8. 「Silver Tree の豆の選び方ガイド（3 商品比較）」
9. 「個人でコーヒー屋を始めるなら、最初に揃えるもの」
10. 「1 週間ドリップバッグセットの楽しみ方」
```

---

### 静止画（柱③ 商品選び方ガイド）— ブランドの蓄積

**目的：** プロフィールページが「ブランドのアーカイブ」になる。初めて来た人が世界観を把握できる。

**写真テーマの 3 パターン：**

| パターン | 内容 | キャプションの型 |
|---------|------|----------------|
| A：商品写真 | 豆・ドリップバッグ・ギフトセット | 商品の説明 + BASE CTA |
| B：プロセス | 焙煎中・ドリップ中の手元 | 工程の一言解説 |
| C：出店シーン | マルシェ・軽バン・野外 | 場所の空気感を一言で |

**静止画の CTA（商品投稿のみ）：**
```
キャプション最後に：
「—— プロフのリンクから BASE で購入できます」
```

### Stories（日常 + 購買導線）

**週のリズム：**

| 曜日 | Stories の内容 |
|------|---------------|
| 月 | 「今週の焙煎予定」（テキストのみ） |
| 水 | 仕込み・準備の写真 1〜2 枚 |
| 木 | 商品紹介 + **Link Sticker（BASE へのリンク）** |
| 土 | 出店・焙煎のリアル写真 |
| 随時 | 注文が来たとき / いいコメントをもらったとき / 面白い出来事 |

**木曜 Stories は購買 CTA の日（週 1 回）:**
```
写真：商品のアップ
テキスト：「今週焙煎した○○です。プロフから購入できます」
Link Sticker：BASE 商品ページへのリンク
```

## キャプションの書き方（更新版）

旧戦略では「回復・疲れた人」のトーン。
新戦略では「焙煎士として語る」トーン。

**旧（使わない）：**
```
疲れた日に、4分だけコーヒーを淹れてほしい。
香りが、少しだけ楽にしてくれるから。

#ひとときの4分 #回復
```

**新（使う）：**
```
焙煎直後の豆には大量のCO₂が含まれている。
そのCO₂が酸素から豆を守っている。
だから「鮮度」が香りにそのまま出る。

Silver Tree は注文後に焙煎して出荷しています。

#自家焙煎珈琲 #スペシャルティコーヒー #焙煎士
```

**キャプション構成（基本型）：**
```
1〜2 行目：事実 or 気づき（興味を引く）
空白行
3〜5 行目：補足・解説・体験
空白行
最終行：CTA（商品投稿のみ）
空白行
ハッシュタグ 5〜15 個
```

## ハッシュタグ戦略（更新版）

### 固定タグ（毎回必ず）
```
#自家焙煎珈琲
#スペシャルティコーヒー
#焙煎士
#旅するコーヒー屋
#silvertreecoffeeroaster
```

### 内容別追加タグ

| 投稿種別 | 追加ハッシュタグ |
|---------|----------------|
| 柱①知識 | #コーヒーの豆知識 #ハンドドリップ #コーヒー好きと繋がりたい |
| 柱②リアル | #コーヒー屋 #マルシェ #軽バン #キッチンカー |
| 柱③商品 | #コーヒーギフト #珈琲 #香川コーヒー #おみくじ手紙 |

### 規模別タグの組み合わせ（投稿ごとに）
```
大タグ（100 万件以上）× 1〜2 個
中タグ（10〜100 万件）× 3〜5 個
小タグ（1〜10 万件）× 3〜5 個
```
→ 小タグで TOP 投稿を狙いつつ、中・大タグで露出を広げる

## プロフィール最適化

### BIO 文（現在 → 更新案）

**現在（推定）：**
```
Silver Tree Coffee Roaster
香川 / 旅するコーヒー屋
4分で回復のスイッチ
```

**更新案：**
```
Silver Tree Coffee Roaster 🌲
自家焙煎コーヒー / 香川 東かがわ市
焙煎士のリアルと、豆の話を毎週投稿
コーヒーの選び方 → ハイライトで
▼ 豆を購入する（BASE）
```

### ハイライト構成（5 本）
```
① 豆の選び方（柱③：各商品の特徴を Carousel で）
② 焙煎のこと（柱①：スペシャルティ・焙煎度の解説）
③ 旅の記録（出店・マルシェの写真）
④ 注文方法（BASE の使い方を Stories で）
⑤ 最新情報（直近の出店告知）
```

## X との連携ルール

X で反応が良かった投稿は Instagram に展開する（プラットフォーム横断で効果を重複させる）。

| X の投稿タイプ | Instagram への展開方法 |
|--------------|----------------------|
| 柱①知識投稿（文章） | Carousel に再構成（視覚化） |
| 柱②リアル投稿（文章） | Reels の補足キャプション or Stories |
| 柱③商品投稿 | 商品写真付き静止画として再利用 |

**ルール：X と Instagram で全く同じ内容を投稿しない。**
X は言葉、Instagram は視覚。同じテーマでも切り口を変える。
