## Instagram 投稿手順

### 写真の編集（2分）
- フィルター：Clarendon弱め or なし
- 明るさ：+10程度
- 彩度：-10〜-15（少し落とす）
- アスペクト比：正方形 or 4:5（縦長）

### キャプション型A（静物：コーヒー・手紙・道具）
```
1行目：情景を一言で
（空白行）
2行目：感情か問いかけ
（空白行）
ハッシュタグ3〜5個
```

### キャプション型B（旅・外の写真）
```
1行目：場所か状況
（空白行）
2行目：気づきか独り言
（空白行）
ハッシュタグ3〜5個
```

### 固定ハッシュタグ5つ
```
#ひとときの4分
#旅するコーヒー屋
#自家焙煎コーヒー
#東かがわ
#silvertreecoffeeroaster
```

### 追加ハッシュタグ（内容に合わせて）
```
#手紙 / #ハンドドリップ / #珈琲時間 / #回復 / #香川
```

## Threads 投稿手順

### 基本ルール
- Xより砕けた文体・独り言・途中経過を見せる
- 写真はあってもなくてもOK
- リプライには必ず返す（対話を大切に）

### 型：今日の裏側
```
今日やったこと or 感じたこと
（続きがある感じで終わる）
```

### XとThreadsの違い
| | X | Threads |
|--|---|---------|
| 文体 | 詩的・完成された言葉 | 話し言葉・独り言 |
| 長さ | 1〜5行 | 5〜10行でもOK |
| 感情 | 抑制的 | 少し素直 |
| 写真 | ブランド写真 | スマホそのまま |

## 週のリズム

| 曜日 | Instagram | Threads |
|------|-----------|---------|
| 月 | ストーリーズのみ | 今週の一言 |
| 火 | **フィード投稿** | 任意 |
| 水 | ストーリーズのみ | 制作・仕込みの裏側 |
| 木 | ストーリーズのみ | 任意 |
| 金 | 任意 | 週の振り返り |
| 土 | **フィード投稿** | 任意 |
| 日 | ストーリーズのみ | 任意 |

**最小ルール：フィードは週2回。ストーリーズは写真が来た日だけ。Threadsは週3回。**

## Instagramビジュアルコンセプト（竹内 藍）

### カラールール
| 要素 | 色 |
|------|-----|
| 背景 | オフホワイト #F8F4EE / 木目 |
| アクセント | ウォームブラウン #6B4226 |
| 差し色 | ディープグリーン #2D4A3E |

### 写真構図の3ルール
1. 余白を必ず1/3以上残す
2. 人の「手」を入れると温度が出る
3. 光は自然光のみ（フラッシュ禁止）

### 投稿テンプレート（3パターン繰り返し）
- A：静物（コーヒー・手紙・道具）← 世界観
- B：風景・旅（移動中・マルシェ）← 旅の記録
- C：テキスト画像（言葉だけ）← 深さ

---

## 3プラットフォーム統一感ルール（竹内 藍）

1. **どこでも同じ色** → プロフ写真・ハイライト・投稿すべてにブランドカラー
2. **どこでも同じ言葉** → 「4分」「回復」「旅するコーヒー屋」を各プロフに記載
3. **どこでも余白** → 1投稿1メッセージ
